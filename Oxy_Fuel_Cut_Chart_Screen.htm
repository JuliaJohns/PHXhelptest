Field Definitions

Torch Type

Select the torch that you are using on the cutting table. If a power supply has only one torch available, the Torch Type selection will not be available.

Material Type

This field displays the type of material for this cut chart: mild steel, stainless steel, or aluminum.

Specific Material

This value identifies a custom cut chart.

Fuel Gas

This field shows the fuel gas for the process.

Material Thickness

This field shows the workpiece thickness for the cut chart. Select a different material thickness to change the cut chart.

Tip Size

This field shows the tip size needed for the torch. Select a different tip size to change the cut chart. The cutting tip part number displays below the Tip Size.

Cutting Tip

This field shows the model of cutting tip.

Cut Speed

This field sets the cutting speed (also known as feed rate) for the material type and material thickness of the workpiece.

You can use this field to override the default Cut Speed. The default Cut Speed value (in inches per minute or mm per minute) comes from the cut chart. If you change this option, it reverts to the original cut chart value when the cut chart is loaded.

Kerf

Kerf is the width of the cut that the tool removes as it cuts the material. The CNC automatically offsets the motion path by half the kerf dimension to make sure that the part is cut at the correct size.

High Preheat Time

This value turns on the high preheat gas valve and sets the time for preheating the workpiece before piercing. When you run the part, you can use the Set, Extend, or Release soft keys to change the preheat time. When a part is being cut, use the Set, Extend, or Release soft keys to change the High Preheat Time. To bypass the preheat timer, press Start 2 times.

Pierce Time

This value is the time the Pierce Control output is on before lowering the torch to the cut height.

Moving Pierce Time

This field sets the amount of time that the pierce control output remains on and allows X/Y motion while piercing. The moving pierce allows the molten material from the pierce to be ejected behind the torch.

Creep Time

After piercing, this value is the time that the torch travels at creep speed set on the Machine Setups > Speeds screen. When the Creep Time ends, the CNC accelerates to full cut speed.

Soft Keys

Save Process

This soft key saves the current process settings to the hard drive.

Reset Process

This soft key resets the cut chart to factory defaults based on the process variables selected. The CNC retrieves these settings from the cut chart file ending in .fac.

Save Cut Charts

This soft key copies the current user and factory cut chart files to USB memory stick. User files have an .usr file extension and the factory files have a .fac file extension.

Examples of user and factory file name are Mild Steel-HPR XD-HPR.usr and Mild Steel-HPR XD-HPR.fac.

If you have changed any cut charts, be sure to use this feature to back up your cut charts before you load a cut chart update (cutchart.exe). Contact your OEM or system integrator if you need to update your cut charts.

Load Cut Charts

This soft key loads cut charts from the USB memory stick.

Change Consumables

This soft key shows the torch consumables and their part numbers. For power supplies except XPR, this screen also shows the accumulated cutting time on the plasma nozzle and electrode or oxyfuel torch tip.
