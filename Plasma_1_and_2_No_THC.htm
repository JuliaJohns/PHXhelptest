Field Definitions

Torch Down Time

This value is the time to lower the torch to the workpiece. Set this value to 0 (zero) if you use torch height control, such as the Sensor THC.

Purge Time

This value is the time to allow any impurities or previous cutting gasses to be cleared from the system before starting a new cutting process. Enter this value when performing the first cut after power up and when switching from one cutting gas to another.

Pierce Time

This value is the time the Pierce Control output is on before lowering the torch to the cut height.

Note: When you are using a ProNest part program with XPR, you cannot override this field.

Creep Time

After piercing, this value is the time that the torch travels at creep speed set on the Machine Setups > Speeds screen. When the Creep Time ends, the CNC accelerates to full cut speed.

Note: When you are using a ProNest part program with XPR, you cannot override this field.

Cut Off Time

This value is the time (-1 – 2 seconds) that Cut Control output stays on after a cut and before the arc turns off.

This value turns off the plasma arc before or after the end of the cut to improve edge quality. A negative value turns off the arc before the end of the cutting machine motion. A positive value turns off the arc after motion stops. The Cut Off Time minimizes notches in part edges that can occur when motion stops with the arc on.

Note: When you are using a ProNest part program with XPR, you can override this field.

Retract Delay

This value is the time delay at the end of a cut before the torch moves to the next pierce.

Full Torch Up Time

This value is the time to raise the torch to the travel limit of the lifter. Set this value to zero if you use an automated torch height control such as Sensor THC.

Partial Torch Up Time

Choose ON for Partial Raise to enable Partial Torch Up Time. Enter the time to raise the torch part way along the travel distance of the lifter. If you are using a torch height control such as the Sensor THC, set this value to zero.

Stop Time

This value is the time (0 – 10 seconds) that the CNC delays X/Y motion at the end of a cut before it moves to the next pierce point. This time can be used to retract the torch to avoid tip-ups.

Note: When you are using a ProNest part program with XPR, you can override this field.

Arc Off Time

This value is the time (0 – 2 seconds) that torch motion continues if the arc is lost while cutting. This option allows motion to continue to the next pierce point. Torch motion stops when Arc Off Time ends.

Note: When you are using a ProNest part program with XPR, you can override this field.

Retry on Transfer Fail

This value is the number of times that the CNC tries to fire the torch in the event that the torch fails to ignite.

Transfer Time

This value is the time to attempt ignition of the torch. The CNC confirms ignition by the Cut Sense input. Enable the use of the Cut Sense input by selecting Yes for Arc On Feedback elsewhere on this screen.

Set Arc Current

Enter the amperage needed to cut the material. This option can be used only with plasma systems that communicate with the CNC.

The Set Arc Current value comes from the cut chart. You can override the Set Arc Current value if you enter a different number in this field. If you change this option, it will revert to the original cut chart value when the cut chart is loaded.

Note: With XPR, this field is read-only.

Corner Current Percent

Enter a percentage of the Set Arc Current. This option reduces the arc current to a percentage that is 50% – 100% of the Set Arc Current value. Reduced arc current when cutting corners can improve corner cut quality.

Arc On Feedback

Set this option to ON to use the Cut/Mark Sense input. The CNC doesn’t start motion until the Cut/Mark Sense input activates.

Partial Raise

This setting raises the torch at the end of the cut for the time specified by Partial Up Time.

Torch Down During Cut

This setting specifies whether Torch Down is left on during cutting. Use this parameter for a pneumatic lifter.

Torch Down Between Cuts

This value is the time to raise the torch to the travel limit of the lifter. You should set the Full Torch Up Time to zero if you use a torch height control, such as Sensor THC.

Ignition

This option enables use of the Ignition output for igniting the torch. If your plasma power supply requires a separate ignition signal, set Ignition to ON. If not, set this option OFF. Typically, Hypertherm power supplies do not require this signal.

Soft Keys

Plasma 1 Cut Chart

This soft key opens the Plasma 1 Cut Chart screen for XPR or for HPRXD and other power supplies.

Load Data/Save Data

Use these soft keys to save the settings from the Process screen. The CNC saves the file on either the CNC hard drive or on a USB memory stick. You can then transfer this file to another CNC or save a copy as a backup.

Load Data

This soft key allows you to import a process.

Plasma 1

This soft key opens the Plasma 1 screen where you can customize and control torch motion and the plasma cutting process.

Plasma 2

This soft key opens the Plasma 2 screen where you can customize and control torch motion and the plasma cutting process.

Marker 1

This soft key opens the Marker 1 screen where you can customize and control torch motion and the plasma marking processes.

Note: The XPR cutting processes automatically include marking process settings. The Marker 1 screen and process parameters are not available for XPR, but marking is still supported.

Marker 2

This soft key opens the Marker 2 screen where you can customize and control torch motion and the plasma marking processes.

Note: The XPR cutting processes automatically include marking process settings. The Marker 2  screen and process parameters are not available for XPR, but marking is still supported.

Timing Diagram

This soft key shows the activation and deactivation of I/O for each type of process.

Cutting Tips

This soft key displays cutting tips for the selected process.
