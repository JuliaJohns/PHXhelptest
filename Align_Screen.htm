Align

 

Use this screen to align parts manually on the workpiece.

Navigation: Shape Library > Shape Wizard > Shape Options > Align

This screen has field definitions and soft keys.

Field Definitions

Manual Window

This window, also called the preview window, shows the part or sheet to be cut.

Corner to Align with

This value is the corner of the workpiece to align the part in.

Scrap Clearance

This value is the amount of clearance between the edge of the workpiece and the part the control will add in when moving to the start point of the part.

Skew Adjustment

This setting determines if the control will adjust for workpiece skew when performing the align function.

Skew Reference

This value is the skew reference corner that you move toward and mark a point along the edge. This is only available if Skew Adjustment is On.

Soft Keys

At Corner

Choose this soft key when at the corner of the workpiece you want to align the part in.

At Skew Point

Choose this soft key when at the edge of the workpiece for skew adjustment. This is only available if Skew Adjustment is On.

Manual Options

This soft key opens the Manual Options screen.

Align Wizard

This soft key opens the Align Wizard. The Align Wizard automates the following operations:

  • Enter coordinates for a skewed workpiece on the table.

  • Align parts to a skewed or aligned workpiece.

View Part/View Sheet

This soft key toggles the display of the part in the Preview window.

  • View Part zooms in to show only the part.
  • View Sheet zooms out to show the part in relationship to the entire workpiece. Phoenix shows the sheet dimensions that you entered in the Cutting screen.

Increment Move Speed

This soft key increases the move speed by a set amount.

Decrement Move Speed

This soft key decreases the move speed by a set amount.

Zero Positions

This soft key returns all axes positions to 0 (zero).
