Field Definitions

Hardware

This section shows the current hardware configuration which includes the Processor Type, Processor Speed, Memory Installed, Hard Drive Size, and Hard Drive Free.

Control Information

This section shows the hardware key number, the CNC model number, and serial number.

Software Versions

This section shows the current version of the CNC’s operating system, Phoenix version, and the versions of other software installed on the EDGE Connect.

Software Modules

This section shows software modules installed on the system.

Firmware versions

This field shows the firmware version of the HPRXD connected to the EtherCAT network when there is an active connection to the HPRXD.

Limited Version

This field displays if the CNC is operating on a trial version of the software. The version of software is available for 30 days. Contact your OEM to have this timer reset.

OEM Days Left

Contact your OEM to have this timer reset.

7 More Days

7 More Days is a temporary password that enables all features on the CNC for 7 days, except for ProNest CNC with True Shape nesting, Connect PLC Advanced, and the True Hole Conversion tool.

Soft Keys

Control Information

This soft key opens the Control Information (Diagnostics) screen.

I/O

Choose this soft key and enter the Machine Setups password to open the I/O screen. The I/O screen shows the I/O assignments for general purpose I/O (1 – 512) and analog inputs (1 – 32).  

Oscilloscope

This soft key launches the oscilloscope function.

HPR System

This soft key opens the HPR System screen and shows you the status for the power supply software revision, gas pressure, usage, I/O, and remote tools.

XPR System

This soft key opens the XPR Diagnostics View where you can run gas tests, monitor the plasma power supply, and view diagnostic information for up to two XPR stations. For more details about the XPR Diagnostics View at the CNC, see Cut and Mark with an XPR on an EDGE Connect CNC (809900).

To view XPR error details and monitor XPR diagnostics with your wireless device, see the instruction manual supplied with your XPR plasma power supply.

Touch Screen Calibration

This soft key launches the touch screen calibration utility for tuning screen response.
