Save to Disk

This screen has field definitions and soft keys. Use this screen to save part programs to a USB memory stick, the CNC hard drive, or a mapped network location (if you have enabled mapping drives in the Special Setups screen).

Field Definitions

Save to

Select this option to save the file to the USB memory stick or to a folder on the hard drive. To add or remove a folder, double-click on the touchscreen where indicated.

File name

Enter the name of the file you are saving.

Save Original Text

The EDGE Connect can import part files programmed for other CNCs. When you import one of these files, the Phoenix operating software translates the file into the format used by the Hypertherm CNC.

The Save Original Text option saves the imported part file in its original format instead of in the EDGE Connect CNC format. This selection is not available if you are saving the file to the USB memory stick from the hard drive.

Soft Keys

Load from Disk

This soft key opens a screen where you can load part programs from a USB memory stick or the CNC hard drive.

Save to Disk

This soft key opens a screen where you can save a part program to a USB memory stick or the CNC hard drive.

Resume Last Part

This soft key loads and executes another part program and then returns to the original part program. The part program and position resumes.

Save System Files to Disk

This soft key opens a screen where you can save the system files to a memory stick or to a .zip file.
