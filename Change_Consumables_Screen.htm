Change Consumables

 

The Change Consumables screen for XPR is different than the Change Consumables screen for HPRXD and other power supplies.

  • XPR
  • HPRXD or other power supply

Change Consumables for XPR

The Change Consumables screen for XPR shows the consumables and part numbers for the currently selected process. Consumable life data is not available for XPR.

Soft Keys

Change Instructions

Choose this soft key to learn how to change consumables.

Manual Options

This soft key opens the Manual Options screen, so that you can reposition the torch to change the consumables.

Setups

This soft key only appears when cutting is paused. This soft key opens the Cutting (Setups) screen. The Cutting screen gives you access to the Process, Watch Window setup, Diagnostics, and the password-protected setup screens.

Help

This soft key opens the Phoenix Help window and displays information about the current screen.

Change Consumables for HPRXD and Other Power Supplies

For HPRXD and all power supplies except XPR, this screen shows the consumables and part numbers for the currently selected process along with consumable life data for statistical analysis. This screen can use an output from the CNC to activate an indicator, such as a light or alarm, to tell the operator that a consumable has reached its expected life. This feature allows the operator to change the consumable and avoid a failure that can affect cut quality or damage the torch. The Change Consumables screen can only track consumable life data and offer features related to that data. The CNC cannot detect consumable condition or failures.

Field Definitions

Auto update max consumable life

This feature can be enabled or disabled on the Special Setups screen. When this feature is enabled, it tracks the consumable life beyond the user-defined set point and assigns that maximum value as the new set point. If this feature is disabled, the user-defined set point remains the same until the user changes it manually.

Oxy Fuel Torch 1 – 12

Choose the Oxyfuel torch for which you want to track consumable life.

Plasma Torch 1 – 8

Choose the plasma torch for which you want to track consumable life.

Minutes

This field is the estimated life in minutes of the torch tip, nozzle, or electrode. This value increases to the maximum life or you can enter a maximum value.

Pierces

This field is the estimated life in pierces of the torch tip, nozzle, or electrode. This value increases to the maximum life or you can enter a maximum value.

Arc errors (actual)

Use the Arc Error Counter input to the CNC from the power supply to track arc errors. Refer to the HPRXD Instruction Manual for information on errors reported by the power supply.

Inches or Millimeters

This field is the estimated life in inches or millimeters of the torch tip, nozzle, or electrode. This value increases to the maximum life or you can enter a maximum.

Minutes/Pierce

You can enter a value that the CNC adds to the overall minutes value for each pierce. When you enter this value, it provides a more accurate representation of overall consumable wear. Pierces cause additional wear on the consumables.

Volts/Minute

This value changes the THC Voltage Offset based on the minutes that have elapsed while cutting in Plasma 1 or Plasma 2 cut mode. The CNC adds a small fraction of a volt per minute of cutting to the THC voltage offset to compensate for consumable wear. Volts per Minute applies only to Station 1 or Station 2.

The Volts per Minute value increases the THC voltage offset until you reset the Volts per Minute to 0 and the THC voltage offset to 0.

Volts per Minute is available for use only with the Sensor THC while you cut in Set Arc Voltage mode. If you are cutting in Sample Arc Voltage mode, set the Volts per Minute to 0.

Last Torch Tip Installed

This field shows the date and time when you installed the selected torch tip.

Last Nozzle Installed

This field shows the date and time when you installed the selected nozzle.

Last Electrode Installed

This field shows the date and time when you installed the selected electrode.

Setups

This soft key only appears when cutting is paused. This soft key opens the Cutting screen which gives you access to the Process, Watch Window setup, Diagnostics, and the password-protected setup screens.

Save Database

This field saves the current database to a USB memory stick.

Help
