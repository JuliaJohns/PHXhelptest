Gas Console Inputs or Outputs

This screen shows the current status of the inputs and outputs from the gas console. You can change the state of the outputs on the Power Supply Outputs screen. This screen has field definitions and soft keys.

Field Definitions

Metering Dip Switch 1 – 4

This field displays the status of the gas console metering dip switches.

Select Dip Switch 1 – 4

This field displays the status of the gas console select dip switches.

Soft Keys

Inputs

This soft key opens the Inputs screen for the plasma power supply or gas console.

Outputs

This soft key is password protected and opens the Outputs screen for the plasma power supply or gas console. Operating these outputs can initiate the real-time signal that could, for example, fire the torch.

HPR Information

This soft key opens the HPR Information screen.
