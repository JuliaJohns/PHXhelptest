Field Definitions

Cut Mode

This field specifies the current cut mode: Plasma, Oxyfuel, or Trial. In Trial mode, you can run the part program without cutting.

Kerf

This field specifies the amount of kerf to apply to the current part program.

To view the kerf compensated cut path after entering a kerf value, press the Kerf soft key under the Part Options menu.

Notes:

  • Select a kerf value carefully because this parameter can generate invalid geometries. For example, adding a kerf of 12.7 mm (0.5 inch) to an arc with a radius of 6.35 mm (0.25 inch) results in an invalid cut path.

  • When using ProNest with XPR, you cannot modify the kerf setting in Phoenix. You can modify the kerf in ProNest.

Kerf Variable, Kerf Value

This field creates a kerf variable table that assigns a variable to a kerf value. You can enter up to 200 variables to create a reference table.

This kerf variable can be used within a part program to define the kerf value and as torch parts wear, the kerf value changes. If the kerf variable value is updated as the consumable wears and changes, the new value is called by the kerf variable command with all programs loaded that use the variable.

The EIA-274D part code for left kerf variable is the G43 code. In the following example, G43 D1 X0.06:

·        G43 is the kerf variable setting

·        D1 is the kerf variable, from 1 – 200

·        X0.06 is the selected kerf value

Speeds: Plasma/Trial/Oxy Fuel

This field specifies the speed for the current cut mode. These speeds are saved independently for trialing and cutting. Both speeds are limited to the maximum machine speed. Cut and trial speeds can also be executed by the F-code in a part program.

Note: When using ProNest with XPR, you cannot modify the Plasma Speed setting in Phoenix. You can modify the Plasma Speed setting in ProNest.

Plasma 2 Cut Speed

This field specifies the speed for Plasma 2 process.

Note: When using ProNest with XPR, you cannot modify the Plasma 2 Cut Speed setting in Phoenix. You can modify the Plasma 2 Cut Speed setting in ProNest.

Marker 1 Speed and Marker 2 Speed

This field specifies the speed for the marker 1 and 2 process. These speeds are saved independently for each marker process and are executed through the marker process selection within a part program.

  • Marker 1 is activated by EIA RS-274D M09 and M10, or an ESSI 9 and 10.

  • Marker 2 is activated by EIA RS-274D M013 and M14, or an ESSI 13 and 14.

Notes:

  • The XPR cutting processes automatically include marking process settings. Marker Speed fields are not applicable for XPR when the XPR is used for marking. The XPR does not use the Marker 1 or Marker 2 process, but the XPR still supports marking.

  • When using ProNest with XPR, you cannot override the marker speed field.

Plate Size

This field specifies the dimensions of the current plate. This dimension is used when loading a part to determine if the part will fit on the plate. The dimension is also used for viewing the part in screen view.

Marker Offset 1 – 8

Enter values for up to twelve programmable marker offsets. The machine is offset by this amount at maximum speed when the appropriate marker offset code is detected.

Marker Offset 9

This field is only for use with the Offset IHS parameter on the Sensor THC/Plasma 1 and 2 Process screens. By entering a value for Marker Offset 9, the torch will move from its current position to the marker offset distance, perform IHS, then return to its previous position. An Offset IHS is often used when cutting a pre-pierced workpiece so that the torch doesn’t perform an IHS on a pierce point.

Marker Offset 10 – 12

If values other than 1 are entered for marker offsets 10, 11 or 12, the Align Wizard and CutPro Wizard will automatically allow you to select the laser pointer or the torch for alignment.

Vent Control 1 – 50

Enter rail values for up to fifty programmable zones for fume extraction damper control. Based on the rail position, the vent controls digital outputs activate dampers at the selected zone for increased performance.

Dwell Time

This field specifies the amount of dwell (delay) that is inserted into the current part program when an appropriate RS-274D program block is reached.

Note: Dwell time can be overridden in the part program. For example, in EIA programming a G04 X3 causes a three second dwell to be inserted at the current program block. A G04 without an X-code inserts a dwell with the current Dwell Time parameter.

Arc Radial Error

Arc Radial Error is used to make sure that the starting and ending radial vectors are within tolerance to describe a valid geometry. This field specifies the arc error tolerance to be used when checking the current segment for dimensional accuracy. All ESSI or EIA programs are comprised of lines, arcs, and circles.

Show Traverse Segments

This field allows traverse segment lines (displayed in yellow) to be turned OFF or ON during all part preview displays.

Retain Skew Adjustment

This field retains the last calculated plate skew for all subsequent parts that are loaded. If disabled, any new part that is loaded will remove any previously calculated plate skew.

Mirror Transverse 2

This option appears when you have a Dual Transverse axis (also called Transverse 2) selected in Machine Setups. Select Mirror Transverse 2 to perform mirror cutting where Transverse 2 moves in the opposite direction as the Transverse axis.

Material Thickness

This option changes the display of material thickness in the Cut Chart screen between gauge, fraction, and decimal. This option only displays when the CNC is using English units.

THC Voltage Offsets

This parameter adds or subtracts voltage from the arc voltage set point. Any value you enter for a voltage offset stays in effect until you change it. Use a voltage offset to compensate for electrode wear in the torch. As the electrode wears, the arc voltage set point needs to increase to maintain a consistent cut height.

Phoenix can automatically increase the voltage offset if you have set the Volts/Min parameter on the Change Consumables screen. As time accumulates on the electrode on the Change Consumables screen, Phoenix increases the voltage offset value. To prevent a voltage offset from accumulating automatically, choose Change Consumables from the Main screen, and set Volts/Min to 0.

To reset the Voltage Offset, choose Setups from the Main Screen and set the voltage offset to 0.

Notes:

  • The voltage offset does not reset to 0 if you change consumables or load a new cut chart.

  • The Volts/Min parameter and consumable life data is not available on the Change Consumables screen for XPR.

  • While cutting, pressing the Raise and Lower station buttons on the operator console of the EDGE Pro or EDGE Pro Ti changes the voltage offset for the selected station by 0.5 per button press.

Status and Program Code list

Dwell Override

When this parameter is enabled, embedded dwell G04 X value codes in an RS-274D program override the operator-entered dwell time.

Optional Program Stop

This setting allows overriding of the optional program stop code M01 in the current part program. If enabled, an M01 code operates identically to M00. If disabled, the M01 code is ignored.

EIA I & J Codes

This setting selects absolute or incremental RS-274D programming mode. In Incremental mode, all offsets for X, Y, I and J are relative to the current block. In Absolute mode, all offsets for X, Y, I and J are relative to an absolute reference point unless they are changed by using a G92 (set axis presets) program code.

EIA Kerf Override

If this setting is disabled, all kerf value codes and the Load Kerf Table variable are ignored. This parameter is enabled by default and cannot be changed while the part program is paused. This is useful when a process is used at a cutting machine that is different from the one that was used to create the part program.

EIA G59 Code Override

When enabled, this option allows codes in the part program to select the following:

  • Cut chart process variables (G59 V500 codes for most plasma power supplies and G59 V509 codes for XPR)

  • Cut chart process parameter overrides (V600 codes for most plasma power supplies and process overrides on the M07 line for XPR)

This option is enabled by default, so that part program codes take precedence over cut chart selections made at the CNC. If the G59 Code Override is disabled, the CNC ignores all G59 codes and M07 process overrides in the part program (except for N2 and AR when marking with XPR).

Note: When cutting or marking with an XPR, Hypertherm strongly recommends using a part program created specifically for XPR. Hypertherm does not recommend using ProNest part programs for HPRXD when cutting or marking with an XPR.

EIA M07/M09 HS IHS Override

When this option is enabled, M07 HS and M09 HS codes in the part program force an IHS when the pierce point is within the Skip IHS Distance set on the Process screen for the ArcGlide THC and Sensor THC.

EIA M08/M10 Retract Override

When this option is enabled, M08 RT and M10 RT codes in the part program cause the torch to retract to the transfer height when the cut is complete. When disabled, the torch retracts to the retract height.

EIA F-Code Override

When this parameter is enabled, embedded F-codes in a part program override any operator-entered cut speed.

Speed +/- Affects F-Codes

When this parameter is enabled, the control applies the speed increase/decrease percentage to all embedded F-codes that are encountered in the part program.

EIA Single Decimal Shift

Some programming styles are structured so that the decimal point in the EIA positioning affecting part sizing is assumed. The EIA decimal shift parameter allows the operator to select the location of the decimal point when translating parts by selecting Normal or Single for the correct translation. The selection should be set to Normal unless your part programs have only a single digit to the right of the decimal point.

Process Select Override

When enabled, this feature allows the part program to override the process select input.

Station Select Override

When enabled, this feature allows the part program to override the currently selected station select input.

Auto Torch Spacing Override

When enabled, this feature allows the part program to override the manually selected torch spacing inputs.

G97 Loop Count Prompt

When enabled, this feature posts a message on the screen to enter the number of loops or repeats to be selected when an EIA G97 code without a “T” value is encountered in the part program.

ESAB Multi Torch Support

When enabled, this feature allows ESAB-style ESSI part programs to map codes to specific station selects.

Force G40 Disable Kerf

In a part program, kerf is enabled and disabled using EIA G41/G42 and G40 codes. Standard operation is to disable kerf at the cut off even if the G40 kerf disable is not in a program. With this parameter, you can turn off the “forced” G40 kerf disable if no G40 is used in the program by disabling the parameter.

G40 Used in Simple Shapes

This parameter is used with the Force Kerf Disable parameter. Disable this parameter if you want to omit the G40 code that is normally inserted into a simple shape from the shape library and has not been processed through ProNest CNC.

Auto Start after APA

This parameter is used with the Automatic Plate Alignment feature to allow cutting to begin automatically after completion of the automatic plate alignment.

EIA Code 2 Decimal Shift

Some programming styles are structured so that the decimal point in the EIA positioning affecting part sizing is assumed. The EIA Code 2 Decimal Shift parameter allows you to select the location of the decimal point when translating parts by selecting Normal or Single for correct translation. The selection should be set to Normal unless your part programs require two decimal shift to the right of the decimal point.

M17 and M18 Used as Cut Codes

When enabled, this setting allows the EIA-274D M17 and M18 codes to be used for cut on and off commands.

M76 Rotary Shortest Path

When enabled, this setting disables tilt and rotate software overtravels for dual tilt-style bevel heads and allows the EIA-274D M76 Rotate go to Home command to select shortest path.

When disabled, this setting allows motion by the longest path when homing. This is advantageous for some bevel head designs.

Move to Mark Points

When enabled, this setting allows you to use the Move to Pierce function to move to the Cut On locations and the Marker On locations with the part program.

Parallel Kerf Enable

Instead of moving to an orthogonal index to the kerf width prior to cutting, the torch instead moves to the kerf width during the first cut segment (in parallel or synchronized with cut motion). This shows up as a diagonal move away from the original cut path to the cut path with kerf.

ESSI F Code Decimal Shift

If using ESSI shipbuilding part program codes when this parameter is enabled, there are 2 implied decimal positions for F Codes in English parts and 1 implied decimal position in Metric parts. If this setting is disabled, the ESSI F Code is read directly as is with no decimal shift.

Stop on Single Arc Lost

When this setting is enabled, any Cut Sense input that is lost for longer than the Arc Off time during the cut pauses the part program with a Cut Sense Lost message.  

When this setting is disabled:

  • If a single Cut Sense input is lost, the part program continues and the Cut Sense Lost message does not display.

  • If Cut Sense inputs are lost on all cutting tools for longer than the Arc Off time, the part program pauses with a Cut Sense Lost message.

M03 and M05 Sets Oxy Fuel Mode

This setting allows M03 or M05 codes to switch the Cut Mode to Oxyfuel while the CNC runs the part program. This setting is rarely used in specialty part programs.

Manual Centerline Spacing

When enabled, this setting spaces the tool stations by the distance that you enter for Torch Spacing in the Manual Options screen.

Soft Keys

Cutting

This soft key opens the Cutting screen.

Process

This soft key opens the Process screen.

Disable Control/Enable Control

This soft key disables the drives by turning off the Drive Enable bit for each axis. It also turns off the Drive Enable output if assigned.

Watch

This soft key allows you to set up Watch Windows.

Password

This soft key allows you to enter a password.

Diagnostics

This soft key opens the Control Information (Diagnostics) screen.

Change to Metric Units/Change to English Units

This soft key changes all measurements from English units to Metric, or from Metric units to English throughout the Phoenix software. The changes take place immediately after you choose the soft key, but you must choose Save Changes when you exit the Setups screen.
