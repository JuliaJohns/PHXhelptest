Plasma Cut Chart

This screen has field definitions and soft keys.

Note: The Plasma Cut Chart screen for XPR is different than the Plasma Cut Chart screen described on this page for HPRXD and other power supplies.

Each cutting tool, such as individual plasma power supplies, has its own cut charts.

The default cut charts that are loaded into the CNC at the factory provide values for all remaining parameters for the cut chart and these are displayed on the right of the Cut Chart screen.

Field Definitions

Torch Type

Select the torch that you are using on the cutting table. If a power supply has only one torch available, the Torch Type selection will not be available.

Material Type

This field displays the type of material for this cut chart: mild steel, stainless steel, or aluminum.

Specific Material

This value identifies a custom cut chart.

Process Current

Enter the current set point for the material thickness and material type or specific material selected.

Plasma/Shield Gases

Select the gas types for the shield and for cutting.

Cutting Surface

Choose whether the workpiece will be cut above or below water. This selection is available only for XPR and HPRXD power supplies.

Material Thickness

This field shows the workpiece thickness for the cut chart. Select a different material thickness to change the cut chart.

Preflow Setting and Cutflow Setting

Set the plasma and shield gas preflow and cutflow. Use these settings for plasma systems that support an autogas console. For plasma systems without autogas, these values display only for reference.

Mixed Gas

This field shows the percentages of gases used for mixed-gas cutting processes, such as for stainless steel cutting.

Cut Speed

This field sets the cutting speed (also known as feed rate) for the material type and material thickness of the workpiece.

You can use this field to override the default Cut Speed. The default Cut Speed value (in inches per minute or mm per minute) comes from the cut chart. If you change this option, it reverts to the original cut chart value when the cut chart is loaded.

Kerf

Kerf is the width of the cut that the tool removes as it cuts the material. The CNC automatically offsets the motion path by half the kerf dimension to make sure that the part is cut at the correct size.

Pierce Time

This value is the time the Pierce Control output is on before lowering the torch to the cut height.

Cut Height Delay

This field specifies the amount of time, in seconds, that the torch spends between pierce height and cut height while X and Y motion advances.

Creep Time

After piercing, this value is the time that the torch travels at creep speed set on the Machine Setups > Speeds screen. When the Creep Time ends, the CNC accelerates to full cut speed.

Cut Height

This field determines the height at which the torch cuts the workpiece. This value can be temporarily fine-tuned on the Process screen.

Transfer Height

When the arc transfers to the workpiece, it can be “stretched” to the pierce height. The transfer height is lower than the pierce height because initiating arc transfer at a high pierce height may result in the arc not transferring to the workpiece at all. Enter the Transfer Height as a percentage of the cut height or as an actual transfer height distance.

Pierce Height

This field specifies the height of the torch during pierces. This value can be entered as a percentage of the cut height or an actual pierce height distance. A common guideline is that thicker material requires a higher pierce height.

Set Arc Voltage

In the CNC, each plasma process has an arc voltage set point, called the Set Arc Voltage, for a given material thickness, cut height, cut speed, gas type, and current. When cutting starts, the CNC tracks the actual voltage drop across the arc and compares it to the Set Arc Voltage. When the actual arc voltage is higher or lower than the Set Arc Voltage, the CNC commands the lifter to move the torch down or up.

  • When the actual arc voltage is higher than the arc voltage set point, the torch moves down.

  • When the arc voltage is lower than the set point, the torch moves up.

  • The higher the arc voltage set point, the higher the cut height.

Enter the arc voltage for the material selected. This is part of automatic voltage control (AVC) Auto height. In general, the higher the arc voltage is set, the higher the torch will be from the plate while cutting.

Set Arc Current

This is the value for the plasma arc current. Enter the amperage needed to cut the material. This value also appears on the Process screen. This parameter can be used only with plasma systems that communicate with the CNC.

Soft Keys

Save Process

This soft key saves the current process settings to the hard drive.

Reset Process

This soft key resets the cut chart to factory defaults based on the process variables selected. The CNC retrieves these settings from the cut chart file ending in .fac.

Save Cut Charts

This soft key copies the current user and factory cut chart files to USB memory stick. User files have an .usr file extension and the factory files have a .fac file extension.

Examples of user and factory file name are Mild Steel-HPR XD-HPR.usr and Mild Steel-HPR XD-HPR.fac.

If you have changed any cut charts, be sure to use this feature to back up your cut charts before you load a cut chart update (cutchart.exe). Contact your OEM or system integrator if you need to update your cut charts.

Load Cut Charts

This soft key loads cut charts from the USB memory stick.

Change Consumables

This soft key shows the torch consumables and their part numbers. For power supplies except XPR, this screen also shows the accumulated cutting time on the plasma nozzle and electrode or oxyfuel torch tip.

Send Process to HPR

This soft key sends the cut chart currently shown on screen immediately to the power supply. This soft key appears only when an HPR power supply is selected in the Station Configuration screen.

Cutting Tips

This soft key displays cutting tips for the selected process.

Help

This soft key opens the Phoenix Help window and displays information about the current screen.
