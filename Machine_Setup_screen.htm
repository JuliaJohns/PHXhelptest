Field Definitions

X axis orientation

This field specifies the X axis as the Transverse or Rail axis. The default setting is Transverse.

Up direction

This field defines the machine motion when you press the ­up arrow key in manual mode.

Right direction

This field defines the machine motion when you press the right arrow key in manual mode.

X and Y motor/encoder

Set this field to Normal for EtherCAT cutting machines.

Dual gantry installed

Select Yes to enable the Dual Gantry axis for a second side drive. The Dual Gantry axis is always parallel to the Rail axis.

CBH installed

Select Yes to enable the Oxyfuel contour bevel head (CBH) axis. CBH can only be installed on axis 4. If this selection is not available, make sure that you do not have a Sensor THC assigned to axis 4.

Tilt rotator installed

Tilt Rotator describes an ACXYZ bevel head, which has a rotational axis (C) combined with a tilt axis (A). Choose this option when your cutting system has this type of bevel head installed.

Dual tilt rotator installed

Choose this option when the cutting system has a second ACXYZ bevel head. Choosing this option also enables a second transverse axis that can be used for a second straight torch, or for a chuck to turn round pipe or rectangular tube stock.

Auto home at power up

This field determines whether the contour bevel head (CBH) and tilt or rotate axes automatically go into the homing routine when the CNC powers up.

Follower initially

This field determines whether the CBH and rotate features are always enabled after power up or are enabled through the part program.

Scaled rotator motion

A bevel head does not have a 1:1 relationship between the commanded C angle rotation to the bevel head and the physical output rotation of the bevel head.

Scale factor

This scale factor is used only with bevel heads. The scale factor depends on the bevel head designs. Frequently, this parameter is not needed when proprietary bevel equations have been incorporated into Phoenix. The Scale Factor must be greater than 1.

Dual tilting rotator

Dual Tilting Rotator describes an ABXYZ bevel head which has two tilt axes (A and B). Choose this option when your cutting system has this type of bevel head installed. The Tilt Rotator Installed option must be selected to make the Dual Tilting Rotator option available.

Dual transverse

This option enables the Transverse 2 axis for use with a second cutting station or as an axis for a chuck for holding pipe or tube stock.

Transformation

AC-style bevel head requires transformation equations for the A and C axes. These equations must be supplied by the Channel Partner to Hypertherm. This option does not apply to ABXYZ-style bevel heads.

Table size

Enter this value between the X overtravel switches and the Y overtravel switches. The X and Y dimensions represent the cutting area of the table and limit the workpiece size dimensions set in the Cutting screen.

Key press logging

This option records key press and other related information in a daily log file. Service technicians use this log file to review events before a fault. When key press logging is enabled, the log file is saved to the hard drive so it can be retrieved by transfer to a USB memory stick. Usually this parameter is set to No.

Sensor THCs installed

This field defines the total number of Sensor THC axes assigned to the CNC.

THC installed on axis and input

Select the THC, which axis it is installed on, and the analog input to use for arc voltage feedback.

Ignore torch collision during IHS

This option allows the system to ignore input about a torch collision during the initial height sense cycle. This can be helpful for some THC mechanics.

Auto torch spacing

Select Yes to enable the automatic torch spacing feature. This feature uses embedded part program codes and dedicated outputs to control individual torch stations to set spacing and clamping of direct and mirrored cutting.

Minimum torch spacing

If Manual Centerline Spacing is disabled in the Cutting screen, the Minimum Torch Spacing value accounts for the width of each torch station.

Minimum Torch Spacing helps to make sure that the torch stations do not collide during Torch Spacing in the Manual Options screen. Otherwise, Phoenix does not use the Minimum Torch Spacing value.

Soft Keys

Machine

This soft key opens the Machine Setups screen.

Speeds

This soft key opens the Speeds screen.

Digital I/O

This soft key opens the Digital I/O screen.

Analog I/O

This soft key opens the Analog I/O screen.

EtherCAT

This soft key launches Hypertherm EtherCAT Studio for network setup and creation of the Phoenix.xml file and network troubleshooting.

Axes 1 – 6

This soft key opens the Transverse screen which allows you to navigate to the Rail, Sensor THC, Rotate, and Tilt screens if available. Phoenix only shows soft keys for the axes that the system supports.

Axes 7 – 12

This soft key opens the Transverse 2 screen. You can navigate to the Rotate 2, Tilt 2, and additional Sensor THC screens if available. Phoenix only shows soft keys for the axes that the system supports.

Laser Mapping

Choose this soft key to set Laser Mapping parameters. Laser Mapping is a feature of Phoenix that improves the linear positioning accuracy of a cutting table if there are mechanical inaccuracies on the axes.

A laser interferometer records a number of positions along the rail, transverse, and dual gantry axes. The CNC uses the data from the interferometer to compensate the commanded position to each drive axis and reduce position errors to 0.
