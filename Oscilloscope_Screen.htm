Oscilloscope

Use the Oscilloscope screen to log I/O, servo output voltage to the drives, analog inputs, and drive status while the CNC is operating. The grid represents the time rate in which the function is recording data. You can set up the Oscilloscope to help you understand an issue with an input or output or to log a function and then provide a visual representation of the log file.

This screen has field definitions and soft keys.

Field Definitions

Digital Inputs and Outputs

Double-click on the I/O point you want to record. Remove the I/O point by pressing the keyboard space bar.

Analog Inputs and Outputs

Double-click on the I/O point you want to record. Remove the I/O point by pressing the keyboard space bar.

Status

Choose the status bits you want to record. Phoenix has status bits for certain features, such as drive enables, torch height control, and tracking. You can view the state of status bits in the Oscilloscope screen and the Watch Window.

Start

Select when you want the Oscilloscope to start recording.

Stop

Select when you want the Oscilloscope to stop recording.

Rate

Select the time intervals used by the Oscilloscope to record the data.

Soft Keys

Record

This soft key starts recording the file.

Stop

Choose Stop to end recording or file viewing.

Pause

This soft key temporarily pauses recording or file viewing.

Play

This soft key starts the file for viewing.

Fast Forward

This soft key speeds up the file for viewing.

Load

This soft key loads an Oscilloscope file from a selected device.

Save

This soft key saves the current Oscilloscope file to a selected device.

Reset Channels

This soft key clears your input and output selections.
