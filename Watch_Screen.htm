Watch Window

The Watch Window appears on the Main screen. It provides feedback during cutting, including position, process data, I/O state, cutting speed, and so on. You can customize up to 10 Watch Windows. A Watch Window that shows HPR plasma system I/O and status becomes available when you select HPR in the Station Configuration screen.

Note: To monitor XPR diagnostic information in Phoenix, select Setups > Diagnostics > XPR System. To view error details and monitor XPR diagnostics with your wireless device, see the instruction manual supplied with your XPR plasma power supply.   

Several options are available to personalize the Watch Window and not all options can be viewed at one time. The options are grouped into two sizes of widgets or icons. Large widgets may be placed into the upper position at the top of the Watch Window or in the middle of the Watch Window. Small widgets are positioned in the lower-left corner of the Watch Window next to the cut information and clock. The cut information and clock at the lower right may not be edited.

Selections in the Watch Window will change slightly based on the CNC I/O configuration and machine options that have been enabled.

This screen has field definitions and soft keys.

Note: When you are using a ProNest part program with XPR, there are some parameters that you can and cannot override from the Watch Window.

Field Definitions

Upper Location

Choose Upper Location and scroll to set the parameter that you want to display in this Watch screen location.

Middle Location

Choose Middle Location and scroll to set the parameter that you want to display in this Watch screen location.

Lower Location

Choose Lower Location and scroll to set the parameter that you want to display in this Watch screen location.

Watch Window options

None

Select None to leave the location blank.

Input/Output

This option shows the current state of inputs, outputs or status information bits. This can be especially useful in debugging gas control sequencing problems. To add or delete a desired input, output, or status bit to the Input/Output list, double-click an item or highlight an item press the + or - keys on the alpha-numeric keypad.

Note: Fixed-function I/O for HPRXD or XPR power supplies and the EDGE Connect TC operator console display in the Input and Output lists after input or output 512.

Speedometer

This option shows cut speed, maximum machine speed, and current machine speed as a gauge.

Digital Speedometer

This option shows cut speed, maximum machine speed, and current machine speed numerically.

Position

In the upper and middle locations, Phoenix can show the positions of 2 axes. In the lower location, Phoenix can show up to 4 axes.

Following Error

This option displays the following error calculated by the drive for that axis. A large following error may indicate that the cut speed may be higher than the cutting system can move. Phoenix can show the following error for two axes in the upper or middle locations. In the lower location, Phoenix can show up to 4 axes.

Command Voltage

This option shows the directional motion command voltage being sent to the amplifier for velocity-type drives. This voltage also equates to the current being commanded for motion in current type drives. Peak voltage can be displayed for a specified amount of time.

Jog Keys

This option shows a directional keypad for manual motion directly from the touchscreen.

System Errors

This option shows errors generated by the CNC and the power supply.

Laser Nozzle

This option shows laser nozzle consumable life. This helps determine when the nozzle should be replaced and tracks nozzle data for statistical process control.

Plasma Torch Tip

This option shows the selected plasma torch tip (1 – 8) consumable life. This helps determine when the torch tip should be replaced and tracks torch tip data for statistical process control.

Plasma Electrode

This option allows the selected plasma electrode (1 – 8) consumable life to be graphically displayed while cutting. This helps determine when the electrode should be replaced and keeping track of electrode data for statistical process control.

HPR Power Supply

Select the HPR power supply that you want to monitor in the Watch Window. Phoenix shows the HPR I/O, and the status of the power supply choppers, current, and gases.

XPR Power Supply

You can monitor XPR I/O in Phoenix by adding the I/O to a Watch Window.

Note: To monitor the status of the power supply choppers, current, and gases for an XPR power supply, use the XPR Diagnostic View at the CNC. Select Setups > Diagnostics > XPR System and choose Plasma Power Supply.

Oxy Fuel Torch Tip

This option shows the selected oxy fuel torch tip (1 – 12) consumable life. This helps determine when the torch tip should be replaced and tracks torch tip data for statistical process control.

Temperature

This option shows the temperature inside the CNC in Fahrenheit or Celsius (selected on the Special Setups screen).

Cut Time

This option shows an estimate of the amount of time it will take to cut the selected part or nest. This window also shows how much time has been completed and how much remains. The estimate of the cut time is based on the complexity of the parts or nest and the cut speed. This window can help you optimize production plans and the use of resources.

Pierces

This option shows the operator how many pierces are needed for the part or nest, how many pierces have been completed, and how many pierces remain. You can use this window to plan consumable changes.

Process Data

The Process Data option shows up to four selected items for a selected cut or marking process. Process timers and status items for Oxy Fuel, Plasma, and Marker may be selected. The process data will only be shown during the current cut process. For example, Plasma 1 process parameters will only be shown in the Watch Window while cutting in Plasma 1 mode.

ProNest Part Program Overrides

When you are using a ProNest part program with XPR, there are some parameters that you can and cannot override from the Watch Window.

Can override:

  • Arc Off Time

  • AVC Delay

  • IHS Start Height

  • Puddle Jump Height

  • Retract Delay

  • Skip IHS Within

  • Cut Off Time

  • Stop Time

Note: If the Check to Automatically Set Parameter option is selected on the Process screen, then you cannot override the parameter.

 

Cannot override:

  • Kerf

  • Plasma Speed

  • Pierce Time

  • Cut Height

  • Cut Height Delay

  • Pierce Height

  • Set Arc Voltage

  • Transfer Height

  • Creep Time

Soft Keys

Watch Number

Choose this soft key and enter the number of the Watch Window you want to set up.

Watch Left/Right Arrows

These soft keys allow you to scroll through the Watch Windows.

Cutting

This soft key opens the Cutting screen.

Process

This soft key opens the Process screen.

Disable Control/Enable Control

This soft key disables the drives by turning off the Drive Enable bit for each axis. It also turns off the Drive Enable output if assigned.

Watch

This soft key allows you to set up Watch Windows.

Password

This soft key allows you to enter a password.

Diagnostics

This soft key opens the Control Information (Diagnostics) screen.

Change to Metric Units/Change to English Units

This soft key changes all measurements from English units to Metric, or from Metric units to English throughout the Phoenix software. The changes take place immediately after you choose the soft key, but you must choose Save Changes when you exit the Setups screen.
