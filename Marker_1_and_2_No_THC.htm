Marker 1 and Marker 2 without Sensor THC Process

Note: The XPR cutting processes automatically include marking process settings. Marker 1 and Marker 2 Process screens are not applicable for XPR, but marking is still supported.

To run the Marker 1 or Marker 2 process, the part program must contain the M36 T3 or M36 T4 process selection code. This code works like the Cut Mode parameter on the Main and Cutting screens. Marker part programs require this code because marker processes are not available in the Cut Mode selection.

In addition:

  • A Marker Offset is set on the Cutting screen so the CNC can position the marker and then reposition the other cutting tool.
  • Because a marking tool is always used with another tool, you will need to use numbered I/O to control the marker.

 

The options on this screen appear when you have selected None or Other for the Lifter in the Station Configuration screen. This screen has field definitions and soft keys.

Navigation: Main screen > Setups > Process > Marker 1 or Marker 2

Field Definitions

Marker Down Time

This value is the time interval (in seconds) to lower the marker tool to the marking height, at the beginning of each mark. When you set a Marker Down Time, it activates the Torch Down Output.

Ignition Time

This value sets the time that the oxyfuel igniter is held on at each ignition of the flame.

Marker On Time

This value sets the time interval (in seconds) that elapses before torch motion starts.

Marker Off Time

This value sets the time interval (in seconds) that elapses before torch motion stops.

Marker Up Time

This value sets the time interval (in seconds) necessary to raise the marking tool to the travel limit of the lifter. When you set the Marker Up Time, it activates the Torch Up output LINK.

Marker Partial Up Time

This value is the time to partially raise the marking tool parallel with the travel distance of the lifter. When you set the Marker Partial Up Time, it enables the Torch Up output.

Arc Off Time

This value is the time (0 – 2 seconds) that torch motion continues if the arc is lost while cutting. This option allows motion to continue to the next pierce point. Torch motion stops when Arc Off Time ends.

Set Arc Current

Enter the amperage needed to cut the material. This option can be used only with plasma systems that communicate with the CNC.

The Set Arc Current value comes from the cut chart. You can override the Set Arc Current value if you enter a different number in this field. If you change this option, it will revert to the original cut chart value when the cut chart is loaded.

Corner Current Percent

Enter a percentage of the Set Arc Current. This option reduces the arc current to a percentage that is 50% – 100% of the Set Arc Current value. Reduced arc current when cutting corners can improve corner cut quality.

Ignition

This option enables use of the Ignition output for igniting the torch. If your plasma power supply requires a separate ignition signal, set Ignition to ON. If not, set this option OFF. Typically, Hypertherm power supplies do not require this signal.

Arc On Feedback

Set this option to ON to use the Cut/Mark Sense input. The CNC doesn’t start motion until the Cut/Mark Sense input activates.

Partial Raise

This setting raises the torch at the end of the cut for the time specified by Partial Up Time.

Down On During Mark

Choose Off to disable the Down On During Mark field. Choose On to enable the Down On During Mark field.

Down On Between Marks

Choose Off to disable the Down On Between Marks field. Choose On to enable the Down On During Mark field.

Cut Control Used for Marking

The CNC uses the Cut Control output to activate the marking tool. When you choose Off, it disables the Marker Control used for marking. When you choose On, it enables the Marker Control used for marking.

Marker Down/Up with Each Marker On/Off

Marker On/Off refers to these part program codes:

  • M09 Marker 1 Enable and M10 Marker 1 Disable

  • M13 Marker 2 Enable and M14 Marker 2 Disable

When the CNC reads the M09 in part program, it turns on the Torch Down output. When the CNC reads the M10 code, it turns on the Torch Up output.

Preheat

When marking with plasma, turn off Preheat. This output is usually used with oxyfuel or zinc markers.

Use Marker 1 Data

Turn ON to set all parameters on the Marker 2 Process screen to the same values as the Marker 1 Process screen, and to disable all controls on the Marker 2 Process screen.
