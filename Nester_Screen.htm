Manual Nester

Use the manual nester for nesting simple shapes. The manual nester does not include any process information with the nested shapes. Use ProNest CNC to create nests that include process information.

In the Preview window, the edge of the workpiece is displayed in dark green. The workpiece size displayed is based on the dimensions that have been entered in the Cutting screen (Main screen > Setups).

The upper-right corner of the Manual Nester screen displays the parts list for the nest in the order the parts will be cut. When you highlight a part in the list, the part displays as red in the main viewing area. In the lower right, you can view and manipulate the part position and orientation information for the selected part program.

Field Definitions

Files

This field displays the part program list for the nest in the order that the parts will be cut.

X Offset

Enter the distance to move the part in the X axis.

Y Offset

Enter the distance to move the part in the Y axis.

Rotate Angle

Enter a positive or negative rotational angle for the selected part.

Mirror X/Mirror Y

Select the check boxes to flip the part horizontally or vertically.

Scale Factor

Enter a size factor to enlarge or reduce the size of the selected part. Phoenix redraws the part and shows the new dimensions. The scale factor must be greater than zero (0) for the X and Y offsets to work.

Soft Keys

Add Part

This soft key allows you to select a part program from a selected source and to add it to the nest.

Remove Part

This soft key removes the selected part from the nest parts list.

Cut Earlier

This soft key moves the selected part to an earlier position in the part cut list. The order in which the parts are cut changes, but the part location in the nest does not change.

Cut Later

This soft key moves the selected part to a later position in the part cut list. The order in which the parts are cut changes, but the part location in the nest does not change.

View Part/View Sheet

This soft key toggles the display of the part in the Preview window.

  • View Part zooms in to show only the part.
  • View Sheet zooms out to show the part in relationship to the entire workpiece. Phoenix shows the sheet dimensions that you entered in the Cutting screen.

Arrow Key (Distance)

This soft key selects 1 of 5 different preset move distances when you use the arrow keys to move parts in the nest. You define these 5 distances in the Nester Setup screen.

Clear Nest

This soft key deletes all of the parts located in the nested parts list from temporary memory.

Setup

This soft key opens the Nester Setup screen.
