Dual Gantry axis

The Dual Gantry Axis is commanded as a separate axis on the CNC that mirrors the output of the Rail axis. Additionally, performance of the Dual Gantry axis is compared to the Rail Axis and additional output command is given to keep the axis in position.

Skew error tolerance

This value equals the position error allowed between the Rail and Dual Gantry axes before causing a fault.

Encoder counts per mm (inch)

This parameter is a position scaling factor used by Phoenix. Refer to the EDGE Connect Setup and Installation Manual (809340) for a general overview of encoder counts. Refer to the drive manufacturer documentation for specific scaling information required by the drives.

Backlash compensation

This setting offsets or compensates for any backlash in the mechanics of the drive system. The maximum value for backlash compensation is 12.7 mm (0.5 in).

Use home limit switch

Select Yes to enable the Home feature for the Dual Gantry axis. You must also assign Dual Gantry Home to an input to use this feature.

Switch offset distance

This distance sets a physical position offset between the Dual Gantry and Rail home switch positions. This allows the CNC to position the two axes accurately for operation and remove any skew of the gantry.

Skew limit

The amount of position error allowed between the Rail and Dual Gantry axes. For example: The Rail axis has 2.54 mm (0.1 inch) of position error, the Dual Gantry axis has 7.62 mm (0.3 inch) of position error, and the Skew Error Tolerance is set to 3.81 mm (0.15 inch). In this case, Phoenix generates a skew error fault because the position error between the Rail and Dual Gantry axes (5.08 mm, 0.2 inch) exceeds the Skew Error Tolerance of 3.81 mm (0.15 inch).

Laser compensation

This function uses position readings gathered with a laser interferometer to correct the actual position of the axes (Transverse, Dual Transverse, Rail, and Dual Gantry only).
