Laser Mapping

Laser Mapping is a feature that improves the linear positioning accuracy of a cutting table if there are mechanical inaccuracies on the axes. A laser interferometer records several positions along the rail, transverse, and dual gantry axes. The CNC uses the data from the interferometer to compensate the commanded position to each axis and reduce position errors to 0 (zero).

To allow Phoenix to compensate for position inaccuracies on an axis, complete the following steps:

  1. Map the axis.
  2. Create a compensation data file for the axis.
  3. Load the data file into the CNC.
  4. Turn on Laser Compensation in the Transverse, Rail, or Dual Gantry screens.
  5. Repeat these steps for each axis you want to map.

Field Definitions

Transverse or Rail First Run

This value is the direction of the first run on the axis. Select the direction of motion from the home position.

Reversal Distance

Enter the distance of travel in the opposite direction at the beginning and end of a run to remove mechanical backlash before mapping. Reverse motion removes mechanical backlash before mapping the axis.

Target Distance

Set the distance between targets. Set this value in larger distances (508 mm, 20 inches) for a rough mapping and in much smaller distances (25 – 50 mm, 1 – 2 inches) for more precise motion compensation.

Number of Targets

Enter the number of positions on each run where the laser interferometer measures the physical position of the axis. The part program that the CNC creates for mapping includes a pause at each of these positions. Enter values between 2 and 1000. Enter a larger number of targets for more precise motion compensation.

Target Dwell

Enter the amount of time that motion pauses at each target to allow the interferometer to take each measurement and record the results. Refer to the instruction manual for your laser interferometer to determine how to set this value.

Mapping Speed

Set the program speed of the part program that the CNC creates for mapping.

Number of Runs

Enter the number of times that the part program runs on an axis. Set this to an even number between 2 and 1000 because the interferometer must travel to the end of the axis and then return. Enter a larger number of runs for more precise the motion compensation.

Soft Keys

Map Transverse

This soft key starts the laser mapping program for the Transverse axis. Phoenix opens the Main screen and loads the mapping program into the Preview window.

Map Rail

This soft key starts the laser mapping program for the Rail axis. Phoenix opens the Main screen and loads the mapping program into the Preview window.
