Shape Manager

The CNC has a built-in Shape Library with more than 68 commonly used shapes. You can edit the size or geometry of the shapes. The shapes in the library are color-coded from green (simplest) to black (most complex).

To load a simple shape:

  1. On the Main screen, choose Shape Manager.
  2. Choose a shape.
  3. Choose OK. Phoenix opens ProNest CNC.
  4. Enter the quantity of parts and select a cut process.
  5. Choose Nest to generate the parts. ProNest CNC includes process expertise (for example, process selection codes) as part of the nest part program.
    Note: Choose Cancel to create a nest without process expertise in the part program.
  6. Choose OK to load the nest and return to the Phoenix Main screen.

Soft Keys

Shape Library

This soft key shows the Shape Library where you can choose a shape.

Text Editor

This soft key opens the Text Editor where you view and change the EIA code that describes the part.

Shape Wizard

This soft key shows the selected shape in a preview window and displays the EIA code next to it. When you select a line of EIA code, the segment that the code describes is highlighted in the Preview window and the segment data definition is shown if the View Segment Data Below check box is selected.

Teach Trace

This soft key opens a screen where you can trace a remnant.

ProNest CNC

This soft key switches from Phoenix to ProNest CNC so that you can nest CAD files.

Manual Nester

This soft key is for nesting basic shapes.
