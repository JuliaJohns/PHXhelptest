Voltage gain

This value controls the relative responsiveness of the entire Sensor THC to maintain the programmed cut height. The higher the value you enter, the more aggressive the response of the lifter. Voltage Gain is the only tuning parameter that affects cutting. The other gains affect raising and lowering the torch and initial height sense motion. Voltage gain can range from 1 to 500. For Sensor THC, the recommended setting is 100.

Servo error tolerance

Servo error, also called following error, is the difference between the commanded motor position and the actual motor position as calculated by the drive. The servo error tolerance is the upper limit of the amount of following error allowed before the CNC faults.   

The amount of servo error tolerance depends on the cutting machine mechanics. Setting the servo error tolerance too low could cause the CNC to fault repeatedly. Setting the tolerance too high may allow inaccurate motion or result in mechanical damage. Set the servo error tolerance to a value higher than the steady-state following error which is reported by the drive.

Stall force tolerance

Stall force is a method of detecting the workpiece during initial height sense. Use stall force when cutting underwater or in other conditions in which ohmic contact is not feasible.

Stall force tolerance is the maximum amount of following error on the THC axis. The CNC starts monitoring the THC axis following error when it reaches the Start IHS Height. The following error accumulates when the torch tip contacts the workpiece and the lifter continues downward movement, but is stopped by the workpiece. When the following error exceeds the stall force tolerance value, the CNC knows the position of the workpiece motion reverses. Motion stops when the amount of reverse travel of the THC equals the Transfer Height.

Note: The recommended Stall force tolerance for Sensor THC is 1.27 mm (0.05 inch).

Encoder counts per mm (inch)

This parameter is a position scaling factor used by Phoenix. Refer to the EDGE Connect Setup and Installation Manual (809340) for a general overview of encoder counts. Refer to the drive manufacturer documentation for specific scaling information required by the drives.

Retry on transfer fail

When IHS completes, the following actions occur:

  1. The torch attempts ignition.

  2. The CNC starts a 10 second timer and waits for the Cut/Mark Sense input (or Cut Sense # input) to turn on.

  3. Finally the CNC displays the status message "Waiting For Arc On."

If the input does not activate within 10 seconds, the ignition process stops, the torch retracts, and the IHS starts again with another attempt to ignite the torch. The CNC repeats this sequence twice.

Slide length

Enter the physical length of the slide. The Sensor THC slide is 239 mm (9.40 inches).

Home

Select the homing method for the THC Axis. If the Home to Switch option is selected, the THC Home (or THC Home 1) input is the Home Switch input. The Sensor THC uses Hard Stop Current Limited.

If you select the Hard Stop Current Limited option, the THC Torque Limit output should be connected to the current limit input of the drive. This reduces torque during homing or initial height sense. Select the option that is compatible with the installed lifter and motor drive.

Current limit %

Set the maximum current output for the operation of the lifter motor, from 0.001 to 100. The default is 50.

Home current limit %

Enter the maximum current during the homing sequence to stall and detect the top of the slide. Set Home Current Limit to 50% of drive capacity, from 0.001 to 100. The default is 10.

IHS current limit %

Set the maximum current during IHS to stall and detect the workpiece, from 0.001 to 100. The default is 5.

Tracking mode

The CNC uses voltage tracking to lock on to the Actual Arc Voltage and compare actual arc voltage to the Set Arc Voltage. You can choose individual tracking mode settings for Plasma 1, Plasma 2, Marker 1, Marker 2, and Laser processes.

For most applications, set tracking mode to Normal. Voltage tracking in Normal mode locks on within 2 volts of the Set Arc Voltage value. Adjust voltage gain to increase lifter responsiveness. In Normal mode, arc voltage accuracy is +/- 1.0 arc volt.

Use Special tracking mode only for applications that require an extra high-speed, high-accuracy response, such as diamond-plate cutting. In Special mode, arc voltage accuracy is +/- 0.25 arc volt.

Average voltage over

Sets the arc voltage sampling rate. Phoenix records the arc voltage every millisecond and averages the arc voltage over the number of milliseconds entered. The average value is compared to the voltage set point and then the command to the lifter is calculated.

For Sensor THC, choose Plasma 1 or 2 as the process, then choose Normal for tracking mode. Set Average Voltage Over to 25 ms. Do not use Special tracking mode with the Sensor THC.

Lock on delta limit

This setting creates a lock-on window above and below the lock-on value. The Set Arc Voltage value, within this window, controls the operation of the THC. Enter the number of volts above and below the lock-on value to create the lock-on window, from 0 to 10 volts. The default is 0.

Lock on rate limit

This setting sets the number of consecutive values within the lock-on window that are required to define lock-on. Enter the number of consecutive values within the lock-on window that must be recorded to remain at lock-on.

Voltage tracking range

This value sets the allowable variation in arc voltage from the set point. If the arc voltage exceeds this range, the CNC faults and pauses cutting. Enter the number of volts of allowable arc variation, from 5 to 400. The default is 20.
