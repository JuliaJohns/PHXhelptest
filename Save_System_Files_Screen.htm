Save System Files

This screen has field definitions and soft keys.

You can save the following files to a USB memory stick as a .zip file:

  • LastPart.txt (the last part file)
  • Phoenix.ini.
  • all .xml files (EtherCAT network configuration files)
  • error messages
  • logs

Note: To save key logs, the Key Press Logging feature must be set to Yes in the Machine Setups screen.

To save system files:

  1. Install a memory stick in a USB port.

  2. From the Main screen, choose Files > Save to Disk > Save System Files to Disk.

  3. On the System Files screen, select one or more files from the Select File to Save list.

  4. Choose Save to Disk or choose Save All to Zip File to have the files saved as Phoenix.zip on the memory stick.

Field Definitions

Select File to Save

Select one or more files from the Select File to Save list.

Calendar

The date of any key log files are highlighted on the calendar when selected in Select File to Save.

Soft Keys

Save to Disk

Choose this soft key to save the files to a memory stick.

Save all to Zip file

Choose this soft key to save all files as Phoenix.zip on the memory stick.
