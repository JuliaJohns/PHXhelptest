Transverse 2 axis

The Transverse 2 axis is also called a Dual Transverse axis. It can be used for either a second cutting station or as an axis that turns a chuck for pipe or rectangular tube cutting.

Servo error tolerance

Servo error, also called following error, is the difference between the commanded motor position and the actual motor position as calculated by the drive. The servo error tolerance is the upper limit of the amount of following error allowed before the CNC faults.   

The amount of servo error tolerance depends on the cutting machine mechanics. Setting the servo error tolerance too low could cause the CNC to fault repeatedly. Setting the tolerance too high may allow inaccurate motion or result in mechanical damage. Set the servo error tolerance to a value higher than the steady-state following error which is reported by the drive.

Encoder counts per mm (inch)

This parameter is a position scaling factor used by Phoenix. Refer to the EDGE Connect Setup and Installation Manual (809340) for a general overview of encoder counts. Refer to the drive manufacturer documentation for specific scaling information required by the drives.

Use hardware overtravels

Select whether the cutting machine uses hardware overtravel switches. If the cutting machine uses hardware overtravel switches, the CNC disables feedback and shows an error message if the inputs become active. It is recommended that hardware overtravel switches be installed.

Backlash compensation

This setting offsets or compensates for any backlash in the mechanics of the drive system. The maximum value for backlash compensation is 12.7 mm (0.5 in).

Fault ramp time

This value sets the motion deceleration time after a fault occurs. At the end of fault ramp time, the drives are disabled.

Minimum head spacing

This value sets the minimum distance that is allowed between the Transverse 1 and Transverse 2 axes.

Home

This value sets a known absolute physical position location on the cutting table that is used for referencing manual Go to Home and other motion commands. This is generally performed through activation of a home switch positioned on the axis giving it a known physical location. The axis may be homed to either a home switch or an overtravel switch.

When you enter a homing command, the CNC moves each axis toward its home switch at the Fast Home Speed until the switches activate. Once the switch activates, motion stops and then the axis moves in the opposite direction off of the switch at the Slow Home Speed. The moment that the switch deactivates, the CNC records the position which provides an absolute reference point for future motion commands.

Absolute home position

This value defines the position of the axis when the home switch is detected.

Home offset distance

This value sets an offset distance from the home switch.

Use software travel limits

Software travel limits, placed within the travel limit of the axis before an overtravel switch or hard stop, stop motion before the axis contacts a hard stop. You can select Fault or Fast Decel for the routine to ramp down motion to a stop. Fault causes an immediate stop. Fast Decel uses the Fast Deceleration rate set on the Speeds screen to ramp down motion.

The blue positions show the software travel limits.

Maximum travel limit

If Software Overtravels are enabled, enter the maximum travel of the axis.

Minimum travel limit

If Software Overtravels are enabled, enter the minimum travel of the axis.

Mirrored marker offsets

This setting allows preset tool offsets to be performed as mirrored motion for the Transverse 2 axis.

Rotating transverse

Select Yes to enable Transverse 2 as a rotating axis for pipe or tube cutting.

Laser compensation

This function uses position readings gathered with a laser interferometer to correct the actual position of the axes (Transverse, Dual Transverse, Rail, and Dual Gantry only).
