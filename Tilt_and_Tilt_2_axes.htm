Servo error tolerance

Servo error, also called following error, is the difference between the commanded motor position and the actual motor position as calculated by the drive. The servo error tolerance is the upper limit of the amount of following error allowed before the CNC faults.   

The amount of servo error tolerance depends on the cutting machine mechanics. Setting the servo error tolerance too low could cause the CNC to fault repeatedly. Setting the tolerance too high may allow inaccurate motion or result in mechanical damage. Set the servo error tolerance to a value higher than the steady-state following error which is reported by the drive.

Encoder counts per revolution

Enter a value that is the number of encoder edges per revolution (rev) of the Contour Bevel Head axis.

You can enter fractional encoder units and the CNC keeps track of these fractions automatically. Encoder counts per rev are equal to the resolution of the encoder multiplied by the encoder lines per rev.

Use the following formula: Encoder Counts/Rev = 4 Counts/Line x 1000 Lines/Rev = 4000

Use hardware overtravels

Select whether the cutting machine uses hardware overtravel switches. If the cutting machine uses hardware overtravel switches, the CNC disables feedback and shows an error message if the inputs become active. It is recommended that hardware overtravel switches be installed.

Absolute home angle

This value defines the position of the axis when the home limit switch is detected.

Home offset angle

This setting allows the user to set an offset angle from the home limit switch.

Homing with overtravel switch

This option activates the Home feature and uses the overtravel switch as its reference. The Home feature is used to set a known absolute physical position location for the tilt axis.

Home direction

This value determines which direction the axes will travel as it moves toward the home limit switch.

Use software travel limits

The Tilt and Tilt 2 axes (A or B axes) have mechanical limits. You can assign both hardware overtravel switches to the limits, and enable software travel limits. A software travel is set at a distance from the hardware overtravel switch to prevent the cutting machine from activating the overtravel switch or hitting a hard stop. Phoenix uses the acceleration rates for the Tilt axis set in the Speeds screen to bring the tilt axis to a stop.

Maximum and minimum travel limits

This value sets the software limit positions from the hardware overtravels in degrees.
