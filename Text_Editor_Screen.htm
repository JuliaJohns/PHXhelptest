Text Editor

The Text Editor screen allows you to write or edit a part program in either EIA or ESSI format. If you have loaded a part, the part program code displays when this screen opens.

  1. With a keyboard installed on the CNC, choose a line of code.

  2. Make changes to existing lines of code or add new lines by pressing Enter on the keyboard.

  3. To edit code without a keyboard, double-click or tap a line of text to edit. The on-screen keyboard opens with the line of code shown. Make changes to the line of code and choose Enter.

  4. Choose OK to save your changes. If you want to save the changes to the hard drive, select Files > Save to Disk.

Soft Keys

Show Original Text

This soft key allows you to view and edit the part program in its original format.

Delete Part

This soft key deletes the current part from the Text Editor so that a new part can be constructed.

Find

This soft key opens the on-screen keyboard and search field for text searches.

Find Next

This soft key finds the next instance of the search term you entered.

Find Previous

This soft key finds the previous instance of the search term you entered.

Replace

This soft key opens the on-screen keyboard for a find and replace search (single instance).

Replace All

This soft key opens the on-screen keyboard for a find and replace search (all instances).
