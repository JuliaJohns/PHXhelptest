Field Definitions

Height Control (Manual/Automatic)

Choose Manual to control the THC with the Raise and Lower station controls on the CNC operator console or Soft Op Con. Before using Manual mode, make sure the torch is close enough to the workpiece to transfer the arc. After cutting starts the torch stays at the height where you positioned it. Manual torch height control enables the IHS In Manual option and disables Sample Voltage.

Choose Automatic to allow Phoenix to control the height of the torch based on arc voltage feedback and the settings in this screen.

IHS In Manual

If this setting is ON, Initial Height Sense (IHS) and the sequence of operations is automatic, but the torch height is not controlled by measured arc voltage.

If this setting is OFF, all operations are controlled manually.

Sample Voltage

When this setting is ON, the THC measures the voltage at the end of the AVC Delay and uses the voltage as a set point for the remainder of the cut.

When this setting is OFF, the Set Arc Voltage is used as the set point for torch height control.

Set Arc Voltage

In the CNC, each plasma process has an arc voltage set point, called the Set Arc Voltage, for a given material thickness, cut height, cut speed, gas type, and current. When cutting starts, the CNC tracks the actual voltage drop across the arc and compares it to the Set Arc Voltage. When the actual arc voltage is higher or lower than the Set Arc Voltage, the CNC commands the lifter to move the torch down or up.

  • When the actual arc voltage is higher than the arc voltage set point, the torch moves down.

  • When the arc voltage is lower than the set point, the torch moves up.

  • The higher the arc voltage set point, the higher the cut height.

Note: When you are using a ProNest part program with XPR, you cannot override the Set Arc Voltage field.

Set Arc Current

Enter the amperage needed to cut the material. This option can be used only with plasma systems that communicate with the CNC.

The Set Arc Current value comes from the cut chart. You can override the Set Arc Current value if you enter a different number in this field. If you change this option, it will revert to the original cut chart value when the cut chart is loaded.

Note: With XPR, this field is read-only.

Cut Height

This value is the height distance (in inches or millimeters) of the torch from the workpiece during cutting. This value can be temporarily fine-tuned on the Process screen.

You can use this field to override the default Cut Height. The default Cut Height value comes from the loaded cut chart. If you change this parameter, it will revert to the original cut chart value when the cut chart is loaded.

Notes:

  • When you are using a ProNest part program with XPR, you cannot override this field.

  • When overriding Cut Height on the Process screen, Pierce Height and Transfer Height automatically adjust in proportion. If you want specific values for Pierce Height and Transfer Height, enter them as absolute values in inches or millimeters after adjusting the Cut Height.

Pierce Height

This value is the height distance (in inches or millimeters) of the torch from the workpiece during pierce. In general, thicker material requires a higher pierce height.

You can use this option to override the default Pierce Height. The default Pierce Height Value comes from the cut chart. If you change this option, it will revert to the original cut chart value when the cut chart is loaded.

Notes:

  • When you are using a ProNest part program with XPR, you cannot override this field.

  • When overriding Cut Height on the Process screen, Pierce Height and Transfer Height automatically adjust in proportion. If you want specific values for Pierce Height and Transfer Height, enter them as absolute values in inches or millimeters after adjusting the Cut Height.

Pierce Time

This value is the time the Pierce Control output is on before lowering the torch to the cut height.

Note: When you are using a ProNest part program with XPR, you cannot override this field.

Cut Speed

This field sets the cutting speed (also known as feed rate) for the material type and material thickness of the workpiece.

You can use this field to override the default Cut Speed. The default Cut Speed value (in inches per minute or mm per minute) comes from the cut chart. If you change this option, it reverts to the original cut chart value when the cut chart is loaded.

Note: When you are using a ProNest part program with XPR, you cannot override this field.

Nozzle Contact IHS

This feature requires that the Height Control be set to Automatic. When ON, the THC uses electrical ohmic contact to sense the workpiece.

When OFF, the THC uses stall force to sense the workpiece. Set to OFF on water tables or painted workpieces because of the unreliability of the electrical contact.

Nozzle Contact Cutting

The THC uses Ohmic contact to sense and retract from the workpiece during marking. This option may be set to OFF for use with water tables, a dirty workpiece, or a cutting or marking process with a very low standoff.

Auto Kerf Detect

To set this parameter, the THC must be in Automatic mode. When Auto Kerf Detect is active, the THC looks for a rapid rise in the measured arc voltage that indicates that the torch is cutting across a previously cut kerf. This parameter temporarily disables the AVC and prevents the torch from diving into the workpiece.

Auto Kerf Detect Voltage

The Auto Kerf Detect Voltage value (1 – 10 volts) should be set high enough to detect normal kerf crossings, but low enough to prevent false kerf detection. In general, the lower the voltage, the higher the detection sensitivity. This parameter is active only if Auto Kerf Detect is ON.

Corner Current Percent

Enter a percentage of the Set Arc Current. This option reduces the arc current to a percentage that is 50% – 100% of the Set Arc Current value. Reduced arc current when cutting corners can improve corner cut quality.

Preflow During IHS

This parameter is used for Rapid Part cutting. When this parameter is active (On), the CNC issues the Start and Hold Ignition signals to the plasma system early to allow the gas preflow to occur while the THC is performing an IHS operation. This reduces the time required to move to the next part and start cutting.

Offset IHS

This parameter enables a remote probe for workpiece detection and initial height sense. If this feature is used, the CNC reads the offset values for Marker Offset 9 set on the Cutting screen (Main > Setups > Cutting). An Offset IHS is often used when cutting a pre-pierced workpiece so that the torch doesn’t perform an IHS on a pierce point. The torch moves the distance of the offset, performs the IHS, and returns to the pierce location. The Z position of the marker offset coordinates height differences between the torch and the probe.

IHS Start Height

This value is the height above the workpiece where the THC starts the initial height sense process. When the torch reaches this distance above the workpiece the following actions occur:

  • Speed slows from Maximum THC Speed to Fast IHS Speed.

  • THC Torque Limit and Nozzle Contact Enable outputs turn on.

  • The CNC monitors the Nozzle Contact Sense input. This input activates when the torch touches the workpiece, so the CNC knows the height of the workpiece.

  • The CNC monitors the axis following error which the CNC compares to the stall force. When the following error exceeds the stall force, the CNC knows the height of workpiece.

  • Setting: 2.54 – 50.8 mm (0.1 – 2.0 in)

Note: When you are using a ProNest part program with XPR, you can override the IHS Start Height field.

Skip IHS Within

This setting disables initial height sense at pierce points if IHS falls within the selected distance. This setting increases cutting productivity. The distance is measured from the end point of the cut segment to the next pierce point. Choose the check box to automatically set this parameter to the default value.

If you de-select the check box, you can enter a different Skip IHS Within value here.

Note: When you are using a ProNest part program with XPR, you can override the Skip IHS Within field.

Transfer Height

When the arc transfers to the workpiece, it may be “stretched” to the pierce height. Enter the Transfer Height as a percentage of the cut height (or as an actual transfer height distance for XPR). The percentage can be 50%– 400% of Cut Height; commonly it is 150%.

Choose the check box to automatically set this parameter to the default value. If you de-select the check box, you can enter a different Transfer Height value here.

The transfer height is lower than the pierce height because initiating arc transfer at a high pierce height may cause the arc to not transfer to the workpiece.

Notes:

  • When overriding Cut Height on the Process screen, Pierce Height and Transfer Height automatically adjust in proportion. If you want specific values for Pierce Height and Transfer Height, enter them as absolute values in inches or millimeters after adjusting the Cut Height.

  • When you are using a ProNest part program with XPR, you cannot override the Transfer Height field.

Puddle Jump Height

This value determines the height above the workpiece where the torch is raised after pierce and before lowering to Cut Height so that the torch clears the top dross puddle that can form during the pierce. Enter a percentage of the Cut Height. The torch remains at this height until the Cut Height Delay has elapsed. The percentage can range between 50% to 500% of Cut Height. If you are not using Puddle Jump Height, set this parameter to 100%.

Note: When you are using a ProNest part program with XPR, you can override the Puddle Jump Height field.

Creep Time

After piercing, this value is the time that the torch travels at creep speed set on the Machine Setups > Speeds screen. When the Creep Time ends, the CNC accelerates to full cut speed.

Note: When you are using a ProNest part program with XPR, you cannot override the Creep Time field.

Cut Height Delay

This value is the time (in seconds) that the torch remains at the Puddle Jump Height before it transitions to the Cut Height. Cut Height Delay helps the torch to clear the top dross puddle that can form during a pierce. If you are not using Puddle Jump Height, set this parameter to 0.

You can use this field to override the default Cut Height Delay (0 – 10 seconds). If you change this parameter, it reverts to the original cut chart value when the cut chart is loaded.

Note: When you are using a ProNest part program with XPR, you cannot override the Cut Height Delay field.

AVC Delay

This value is the duration (in seconds) to allow the power supply to achieve steady-state operation at the cut height before automatic voltage control begins. When AVC Delay ends, the AVC is enabled for the remainder of the cut. If the THC is in Sample Voltage mode, the arc voltage sample is taken after this delay.

Note: When you are using a ProNest part program with XPR, you can override the AVC Delay field.

Cut Off Time

This value is the time (-1 – 2 seconds) that Cut Control output stays on after a cut and before the arc turns off.

This value turns off the plasma arc before or after the end of the cut to improve edge quality. A negative value turns off the arc before the end of the cutting machine motion. A positive value turns off the arc after motion stops. The Cut Off Time minimizes notches in part edges that can occur when motion stops with the arc on.

Note: When you are using a ProNest part program with XPR, you can override the Cut Off Time field.

Arc Off Time

This value is the time (0 – 2 seconds) that torch motion continues if the arc is lost while cutting. This option allows motion to continue to the next pierce point. Torch motion stops when Arc Off Time ends.

Note: When you are using a ProNest part program with XPR, you can override the Arc Off Time field.

Stop Time

This value is the time (0 – 10 seconds) that the CNC delays X/Y motion at the end of a cut before it moves to the next pierce point. This time can be used to retract the torch to avoid tip-ups.

Note: When you are using a ProNest part program with XPR, you can override the Stop Time field.

Retract Height

This value is the height above the workpiece to which the torch retracts at the end of a cut.

Note: When you are using a ProNest part program with XPR, you can override the Retract Height field.

Kerf Reacquire Time

When Auto Kerf Detect detects a sudden drop in voltage, it activates the Torch Height Disable output for the Kerf Reacquire Time. When the Kerf Reacquire Time elapses, the CNC turns off the Torch Height Disable output and the THC starts to track voltage again.

Note: When you are using a ProNest part program with XPR, you can override the Kerf Reacquire Time field.

Default All Parameters

The CNC can automatically estimate the values for the plasma or marking process. Choose the Default All Parameters soft key to load the estimated values. Under most conditions, these estimated values yield good results. However, you can override any of these values for special conditions. To override an estimated value, clear the check box for that parameter and enter a new value. The CNC displays your entered value in blue.
