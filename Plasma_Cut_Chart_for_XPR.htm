Plasma Cut Chart for XPR

This screen has definitions of process selection fields, selected IDs, parameter fields, and soft keys. The selected IDs and parameter fields are read-only and cannot be modified on the XPR Cut Chart screen.

XPR has its own cut charts. The Plasma Cut Chart screen for HPRXD and other plasma power supplies is different than the Plasma Cut Chart screen for XPR cut chart screen described below. The default cut charts that are loaded into the CNC at the factory provide values for all remaining parameters for the cut chart and these values are displayed on the right of the Cut Chart screen.

Process Selection Fields

Material Type

This field displays the type of material for this cut chart: mild steel, stainless steel, or aluminum.

Material Thickness

This field shows the workpiece thickness for the cut chart. Select a different material thickness to change the cut chart.

Cutting Surface

Choose whether the workpiece will be cut above or below water. This selection is available only for XPR and HPRXD power supplies.

Process Name

Choose the process current and gas.

Note:

  • (Ar) = Argon Assist Pierce (Requires a VWITM  or OptiMixTM  gas console)
  • (ES) = Edge Start

Selected IDs

Record ID

This read-only field is the unique identification number for the record in the cut charts that contains all the parameters needed by Phoenix to execute the part program. The record includes the process IDs, which Phoenix sends to the XPR when executing the part program.

Cut Process ID

This read-only field is the identification number for the process specified in the cut chart. The process contains the parameters the XPR uses for cutting and True Hole cutting.

Marking Process ID

This read-only field is the identification number for the marking process specified in the cut chart. The process contains the parameters the XPR uses for marking.

Parameter Fields

Note: The XPR cut chart parameter fields are read-only and cannot be modified.

Cut Speed

This read-only field shows the cutting speed (also known as feed rate) for the material type and material thickness of the workpiece. The default Cut Speed value (in inches per minute or mm per minute) comes from the cut chart.

Kerf

Kerf is the width of the cut that the tool removes as it cuts the material. The CNC automatically offsets the motion path by half the kerf dimension to make sure that the part is cut at the correct size.

Note: Kerf is a read-only field.

Pierce Time

This value is the time the Pierce Control output is on before lowering the torch to the cut height.

Note: Pierce Time is a read-only field.

Cut Height Delay

This read-only field shows the amount of time in seconds that the torch spends between pierce height and cut height while X and Y motion advances.

Creep Time

After piercing, this value is the time that the torch travels at creep speed set on the Machine Setups > Speeds screen. When the Creep Time ends, the CNC accelerates to full cut speed.

Note: Creep Time is a read-only field.

Cut Height

This read-only field lists the height at which the torch cuts the workpiece.

Transfer Height

When the arc transfers to the workpiece, it can be “stretched” to the pierce height. The transfer height is lower than the pierce height because initiating arc transfer at a high pierce height may result in the arc not transferring to the workpiece at all. The Transfer Height value is read-only and is shown as an absolute value in inches or millimeters.

Note: For XPR, transfer height is independent of cut height.

Pierce Height

This read-only field specifies the height of the torch during pierces as an actual pierce height distance in inches or millimeters. A common guideline is that thicker material requires a higher pierce height. For XPR, pierce height is independent of cut height.

Arc Voltage

This read-only field specifies the arc voltage for the material selected. This is part of automatic voltage control (AVC) Auto height. In general, the higher the arc voltage is set, the higher the torch will be from the plate while cutting.

 Arc Current

This read-only field specifies the amperage value for the plasma arc current needed to cut the material. This parameter can be used only with plasma systems that communicate with the CNC.

Soft Keys

Send Process to XPR

Press this soft key to send the cut chart currently shown on screen immediately to the power supply. This soft key appears only when an XPR power supply is selected in the Station Configuration screen.

Note: Before a process can be sent to the XPR, each of the following conditions must be satisfied:

  • Connected network (field bus running)

  • Enabled station

  • XPR Ready

    • Station must be in one of the following two states: Initial-check (02) or Wait-For-Start (05).

    • Make sure the XPR system is powered ON, is done purging gas, and there are no errors present.

Cutting Tips

This soft key displays cutting tips for the selected process.

Help

This soft key opens the Phoenix Help window and displays information about the current screen.

Load cut charts

This soft key loads cut charts from the USB memory stick.

Change consumable

This soft key opens a screen that shows torch consumables and their part numbers.
