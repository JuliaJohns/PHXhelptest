Remnant Trace

In Remnant Trace mode, you can move the machine to as many positions as needed. There is no limit to the number of positions that you can teach the system. Then Phoenix connects the last position to the first to complete the cut path and create a part. Phoenix saves the part in EIA format and the part can be cut, saved or edited using any of the part options.  

To trace a remnant:

  1. Jog to the point on the Remnant window where you want the trace to begin. Use the joystick or jog keys to move the torch over the workpiece.

  2. Choose First Point.

  3. Jog the torch to the next point and choose Next Point. Repeat this step until you have traced all but the final point.

  4. When the torch is over the last point you need to trace, choose Last Point. Phoenix draws a line between this point and the first point to close the remnant.

  5. Choose OK. Phoenix returns to the Main screen with the remnant shown in the Preview window.

  6. Choose Files > Save to Disk.

  7. Select a folder for the new remnant file from the Save to drop-down list.

  8. Enter a file name and choose OK.

Soft Keys

First Point

This soft key allows you to select the first point in remnant.

Next Point

This soft key allows you to select the next point in the remnant. Use this soft key as many times as you need to outline the remnant shape.

Last Point

This soft key allows you to select the last point you need to trace. Trace Remnant draws a line between this point and the first point to close the remnant.

Select Teach Trace Mode

This soft key allows you to use Teach Trace mode to create remnants.
