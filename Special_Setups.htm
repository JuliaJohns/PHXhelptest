Special Setups

In the Special Setups screen, you can do the following:

  • Set the local date, time, and language.
  • Set the user level to control what features are displayed for the operator.
  • Give permissions for files and folders.
  • Turn certain messages on and off.
  • Change the passwords to open the Machine Setups, Station Configuration, and Special Setups screens.

Date and time

Enter the date and the local time. The time displays on the Main screen.

Display time

Choose whether to show the time in 12-hour (AM/PM) or 24-hour format.

Keyboard

Select Installed to use an external keyboard on the CNC. Select Not Installed to use the on-screen keypad.

Touchscreen

Choose Not Installed if you are not using a touchscreen and would like to see keyboard shortcuts on the soft keys instead of icons. Phoenix functions the same way regardless of how you set this parameter.

Mouse pointer

Choose On to enable use of a mouse.

Language

Choose a Phoenix user interface language from the available languages stored on the CNC. Choosing the Phoenix language also chooses the language for the help and manuals.

Note: You must restart the CNC after changing the language.

User level

Choose the user experience level. The level you choose determines which screens and features Phoenix displays.

File extensions

Enter the file extensions of the part files that you will run on the CNC. The default file extension for Phoenix parts is .txt. Hypertherm ProNest parts have the .CNC extension. Refer to your CAM software for the file extension it uses and enter the extension on this screen. Phoenix now uses ProNest CNC to open DXF and other CAD files. Choose the ProNest CNC soft key on the Shape Library screen to open ProNest CNC.

Tools installed

This setting selects the cut processes available to the CNC operator, for example, Plasma 1 and Marker 1. Tools Installed enables the Process screens for the selected cut processes.

Zero positions

This setting determines whether the CNC operator has the ability to reset all axis positions to zero. When disabled, the zero positions soft key is grayed out and not available to the operator.

Oxyfuel cut chart

Selecting Oxyfuel as one of the Tools installed on the Special Setups screen enables cut charts for Oxyfuel processes.

Machine position resets

When enabled, this setting resets the absolute machine position when the Zero Positions soft key is pressed. Only the current incremental motion (part) position is reset to zero. Position information based on homing is not lost.

Auto update max consumable life

If this feature is enabled, it tracks the consumable life values beyond the user-defined set point and assigns that maximum value as the new set point. If this feature is disabled, the user-defined set point for maximum consumable life is not updated.

Latch manual keys

When enabled, the manual motion keys remain on with a single key press. Latching is useful when the cutting table is very large and you need to reposition the tool manually. Latch keys work in the Soft Op Con. To turn on Latch keys, press the Manual key twice.

Arc speed limit

This setting turns off the Speed Limit Check that is performed for arcs larger than 254 mm (10 inches). This compensates for abrupt motion commands caused by non-tangent line arc segments and smooths out motion.

Password

This option changes the Machine Setups screen password. You can use numbers or letters.

Special password

This option changes the Special Setups screen password. You can use numbers or letters.

Station configuration password

This option changes the Station Configuration screen password. You can use numbers or letters.

Tangent angle

This value sets the degree of the tangent angle for motion control. Segments within a part that intersect at angles greater than the selected tangent angle will decelerate to zero or to the minimum corner speed. Segments within a part that intersect at angles less than or equal to the selected tangent angle do not decelerate unless the next segment is a speed-limited arc.

Force simulation

Set Force simulation to Yes if you are running a version of Phoenix in simulation mode on a laptop computer, for example. The EtherCAT network does not work when Force Simulation is set to ON.

Status/Feature

To change the status of a software feature, double-click or press Space.

Add or delete folders

This feature restricts the operator's ability to add or delete folders on the CNC or the host computer.

Delete files

This feature restricts the operator's ability to delete files from the parts folder on the CNC or the host computer.

Map drives

This feature restricts the operator's ability to map to external drives from the CNC through the network.

Configure watch

This feature restricts the operator's ability to change the items in the Watch Window.

Add, remove, or change processes

This feature restricts the operator’s ability to add, remove, or change a cut chart.

Torch Spacing

This feature turns on manual torch spacing in the Manual Options screen. This feature allows the operator to control torch positioning before running a part program in a multiple-torch system.

Status/Message or Wizard

To change the status of a message, double-click the message or press Space.

Ready to start cutting

When enabled, the Ready Message feature displays a message after you press the Start button.

Kerf too large warning

This message notifies you that a conflict between the cut paths and the current kerf value for a part has been detected and that some detail could be lost when the part is cut.

Homing Must Be Performed

This feature prompts you to home the selected axis before motion begins. The selected axis must have homing enabled on the axis setup screen.

Values Have been Changed

This feature displays as a confirmation when changes to values on the current configuration screen are detected. This ensures that changes are not made in error when exiting the screen.

Home Torch Height Control

This feature prompts you to home the Sensor THC axis at power up and before motion begins.

Ready to Final Align

This feature appears as part of the skew alignment function and displays just before final motion to position the tool begins.

Part Larger than Plate

This feature displays when the dimensions of the part that have been loaded exceed the selected plate dimensions.

Cut Chart Data Has Changed

This feature displays as a confirmation when changes to the values on the current Cut Chart screen are detected. This ensures that changes are not made in error when exiting the screen.

Save Part for Rush Job

This feature enables or disables the Resume Part prompt for Rush Job Interrupt.

Automatic Align Wizard

This feature enables or disables the automatic availability of the alignment wizard on the Align screen.

Automatic CutPro Wizard

This feature enables or disables the automatic availability of the CutPro wizard on the Main screen. If availability of the wizard is enabled, it appears 10 seconds after the Main screen displays after power-up.

Start Cut from CutPro Wizard

This feature allows or prevents cutting to start directly from the CutPro Wizard. If this is disabled, the operator must use the Start button on the console to begin the cut.

True Hole Verification

On a True Hole cutting system, Phoenix can perform a series of checks to ensure that the correct settings are made for True Hole cutting. You can enable or disable these checks individually. When enabled, Phoenix can automatically correct many of these settings.

True Hole I/O Verification

Checks for these outputs:

  • Pierce Control

  • Pierce Control 1 – 20

  • Pierce Control THC 1 – 4

True Hole Setups Verification

Checks for the following parameters and settings:

  • EIA Kerf Override – Enabled

  • EIA F-code Override – Enabled

  • EIA G59 code Override – Enabled

  • Process Select Override – Enabled

  • Parallel Kerf Enabled – Disabled

  • Arc Off Time ≥ .05 seconds

  • Cut Off Time = 0

  • If Auto Select Inputs AND Manual Select Inputs are assigned: Station Select Override – Enabled

  • Pierce on with Cut On – Yes

True Hole OpCon Verification

Checks the operator console switches:

  • If any Auto Select inputs are assigned, then at least one station must be set to Program.

  • Program speed set to 100%.

Invalid process current selected

In the cut chart, you have selected a process current that is not supported.

Jog key watch warning

An option for Watch Window setup is to display manual-motion jog keys on the CNC touchscreen. However, if the CNC is installed in an environment where water droplets could come into contact with the touchscreen, unintended motion could occur. This warning displays when setting up a Watch Window with jog keys.

Verify lens installed and Verify nozzle installed

This option is not used in Phoenix Version 10.0. For laser processes, these messages can appear after you have changed the cut chart and the new cut chart requires a different nozzle or lens.

Unable to load setups

When enabled, Phoenix displays this message when it cannot load the Phoenix.ini file.

Ready to move

When enabled, this message displays before any motion is executed by the EDGE Connect.

Ignore HPR Link Fail

The HPR Link Fail message appears under different conditions, including when the HPR system is turned OFF. You can hide the HPR Link Fail message when the HPR is OFF by enabling Ignore HPR Link Fail.

Soft Keys

Save Setups

This soft key saves the current CNC setup file (Phoenix.ini) to the USB drive or hard drive. After installation of the CNC on the cutting table (or if any setup parameters are changed), save the current setup file to both the hard drive and USB drive for future use and as a backup.

Load Setup

This soft key loads a setup file onto the CNC. A window displays where you can select the location and enter the file name.

Update Software

This soft key updates Phoenix software from a USB memory stick from the file PhoenixSuiteInstaller.exe.

Update Help

This soft key updates the CNC help files from a USB memory stick from the file Help.exe.

System

This soft key opens the System screen which shows table manufacturer information and CNC performance information.

Restore Last Version

This soft key restores the CNC to the previous version of Phoenix software.

Update PLC Program

Allows you to load a new program for PLC Connect or PLC Connect LT.

  1. Create a copy of the original PLC program file before performing the update.

  2. Choose Setups > Password > Special Setups > Update PLC Program.

  3. Navigate to the folder where the new PLC program is located.

  4. Answer Yes to the prompt "Save Current retentive Data?" This data is saved in the folder located at C:\Phoenix\PLC.

  5. Restart Phoenix after you update the PLC program.

Update Manuals

To get the newest manuals available from Hypertherm, visit www.hypertherm.com and choose the Documents Library link.

  1. In the Documents Library, choose Product Type, then select a Product Name. For example, choose MAXPRO200 to display a list of manuals and other literature available for that product.

  2. Choose the Manuals link and click to download a manual file.

  3. Save the file in the root folder of a USB memory stick. Do not change the name of the file from its name in the Documents Library. It will have a name like 807700r0.pdf.

To load the manual onto your CNC, follow these steps. You can load more than one manual at a time onto the CNC as long as the files are in the root folder of the memory stick.

  1. Insert the memory stick containing one or more Hypertherm product manuals into a USB port on the CNC.

  2. Choose Setups > Password, and enter updatemanuals (all lowercase, one word). The CNC copies the manuals from the memory stick to the hard drive.
