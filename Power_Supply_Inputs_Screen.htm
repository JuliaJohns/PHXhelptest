Power Supply Inputs and Outputs

These screens show the current status of the inputs or outputs from the plasma power supply. You can change the state of the outputs on the Outputs screen for the plasma power supply.

Soft Keys

Inputs

This soft key opens the Inputs screen for the plasma power supply or gas console.

Outputs

This soft key is password protected and opens the Outputs screen for the plasma power supply or gas console. Operating these outputs can initiate the real-time signal that could, for example, fire the torch.

HPR Information

This soft key opens the HPR Information screen.

Note: To view XPR information in Phoenix, select Setups > Diagnostics > XPR System and select Plasma Power Supply.
