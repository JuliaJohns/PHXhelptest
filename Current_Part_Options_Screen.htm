Current Part Options

Use this screen to customize the layout of the current part. This screen has field definitions and soft keys.

Scale Factor

Rotate Angle

Mirror X/Mirror Y

 

Field Definitions

Scale Factor

Enter a size factor to enlarge or reduce the size of the selected part. Phoenix redraws the part and shows the new dimensions. The scale factor must be greater than zero (0) for the X and Y offsets to work.

Rotate Angle

Enter a positive or negative rotational angle for the selected part.

Mirror X/Mirror Y

Select the check boxes to flip the part horizontally or vertically.

Soft Keys

Kerf

This soft key turns kerf path display ON and OFF.

Repeat

This soft key opens the Repeat screen.

Align

This soft key opens the Align screen where you can launch the Align Wizard, align the part to one corner of the plate, or accommodate skewed plates when aligning the part.
