Field Definitions

Ignition Time

This value sets the time that the oxyfuel igniter is held on at each ignition of the flame.

Low Preheat Time

This value sets the time interval in which the Low Preheat Control output is on. When you enter a Low Preheat Time, it activates the Low Preheat Gas Valve.

High Preheat Time

This value turns on the high preheat gas valve and sets the time for preheating the workpiece before piercing. When you run the part, you can use the Set, Extend, or Release soft keys to change the preheat time. When a part is being cut, use the Set, Extend, or Release soft keys to change the High Preheat Time. To bypass the preheat timer, press Start 2 times.

Staged Pierce

This feature performs the pierce in a timed progression of outputs that can control oxygen pressure. When you select this option, the Staged Pierce Time 1 â€“ 3 parameters replace the Pierce Time, Moving Pierce Time, and Creep Time parameters. Set the timer for each staged pierce output.

Pierce Time

This value is the time the Pierce Control output is on before lowering the torch to the cut height.

Moving Pierce Time

This field sets the amount of time that the pierce control output remains ON and allows X/Y motion while piercing.

Creep Time

After piercing, this value is the time that the torch travels at creep speed set on the Machine Setups > Speeds screen. When the Creep Time ends, the CNC accelerates to full cut speed.

Primary Torch Up Time

This field sets the amount of time to raise the torch after completing a cut. After each cut, the torch continues to raise until the Primary Torch Up Time ends or until the lifter reaches a limit switch, which activates the Torch Up Sense input.

Primary Torch Down Time

This field sets the amount of time of time to lower the torch after the torch ignites. At the beginning of each cut, the torch continues to lower until the Primary Torch Down Time ends or until the lifter reaches a limit switch that activates the Torch Down Sense input.

Pierce Torch Up Time

This field sets the amount of time to raise the torches after it pierces the workpiece, to clear a pierce puddle.

Pierce Torch Down Time

This field sets the amount of time to lower the torch to cut height.

Cut Off Time

This field sets the amount of time that the Cut Control output remains on at the end of a cut. Cut Off Time allows the torch to finish its cut and removed any lag (a slight angle that is created when the flame meets metal and bends). Use this parameter to add time for the flame to become perpendicular before it is turned off.

Bleed-off Time

At the end of a cut, this value is the time that the torch pauses to purge gas before the torch travels to the next cut. The Bleed-off Time can overlap the Primary Torch Up Time.

Cut Control Delay

While the workpiece is being pierced, this field sets the time that the CNC waits before it activates the Cut Control output.

Lifter Low Speed

Lifter Low Speed works only with multiple torch systems. This timer turns on with the Torch Up and Torch Down outputs and shuts off after the Lifter Low Speed time ends. The value for the Lifter Low Speed timer should be less than the value for the Primary Torch Up Time and Primary Torch Down Time.

Ignitors

Set Ignitors to No and Ignition Time to 0 to activate the Low Preheat Control output at the end of a cut. Set Ignitors to Yes to turn off the Low Preheat Control output and re-ignite the flame at each next pierce point.

Low Preheat During Cut

When set to Yes, this feature sets the Low Preheat Control output to ON during cutting.

Preheat During Cut

When set to Yes, this option sets the Preheat Control output to ON during cutting.

Torch Down During Cut

This setting specifies whether Torch Down is left on during cutting. Use this parameter for a pneumatic lifter.

Soft Keys

Oxyfuel Cut Chart

Choose Oxyfuel Cut Chart to go to the Oxyfuel Process screen where you can view, customize, and control torch motion and processes for oxyfuel cutting/marking.

Load Data/Save Data

Use these soft keys to save the settings from the Process screen. The CNC saves the file on either the CNC hard drive or on a USB memory stick. You can then transfer this file to another CNC or save a copy as a backup.

Load Data

This soft key allows you to import a process.

Oxyfuel

This soft key opens the Oxyfuel screen where you can customize and control torch motion and the processes for oxyfuel cutting/marking.

Plasma 1

This soft key opens the Plasma 1 screen where you can customize and control torch motion and the plasma cutting process.

Plasma 2

This soft key opens the Plasma 2 screen where you can customize and control torch motion and the plasma cutting process.

Marker 1

This soft key opens the Marker 1 screen where you can customize and control torch motion and the plasma marking processes.

Note: The XPR cutting processes automatically include marking process settings. The Marker 1 screen and process parameters are not available for XPR, but marking is still supported.

Marker 2

This soft key opens the Marker 2 screen where you can customize and control torch motion and the plasma marking processes.

Note: The XPR cutting processes automatically include marking process settings. The Marker 2  screen and process parameters are not available for XPR, but marking is still supported.

Timing Diagram

This soft key shows the activation and deactivation of I/O for each type of process.

Help

This soft key opens the Phoenix Help window and displays information about the current screen.
