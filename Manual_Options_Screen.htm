Field Definitions

THC Jog Speed

When the lifter is a Sensor THC, select Slow IHS, Fast IHS, or Full Jog Speed to set the speed for the Raise Torch and Lower Torch soft keys on this screen.

Laser Pointer

Use the User Defined offset to manually offset a laser point by entering values for Transverse and Rail axes. You can assign a laser pointer offset to separate cutting processes using the Marker Offsets on the Cutting screen. After you have defined a laser point offset for a cutting process, the offset will appear in the Manual Offset drop-down list on this screen.

  • User Defined: Enter an offset for the Transverse and Rail axes.

  • Laser pointer to Plasma 1: For the Plasma 1 process, enter an offset for Marker Offset 10 on the Cutting screen.

  • Laser pointer to Plasma 2: For the Plasma 2 process, enter an offset for Marker Offset 11 on the Cutting screen.

  • Laser pointer to Oxy Fuel: For the Oxyfuel process, enter an offset for Marker Offset 12 on the Cutting screen.

  • Choose the Manual Offset soft key on the Manual Options screen to move the tool the offset distance.

Torches

Select the number of torches to space (2 – 20).

Spacing

Enter the distance to space the torches. Choose the Space Torches soft key on this screen to move the torches apart by the spacing distance.

Soft Keys

The soft keys shown on this screen depend on the axes defined in the machine setup. You may not see all of the soft keys described below.

Raise Torch

This soft key raises the torch until the Torch Up Sense input activates. If a Sensor THC is installed, Phoenix uses the THC Jog speed selection on this screen for the speed.

Lower Torch

This soft key lowers the torch until the Torch Down Sense input activates. If the Torch Down output has been enabled to stay on during the cut in plasma setups, press the Lower Torch soft key to keep the torch in the down position until the Lower Torch soft key is depressed a second time. If a Sensor THC is installed, Phoenix uses the THC Jog speed selection on this screen for the speed.

Manual Offset

This soft key moves the laser pointer the offset distance selected on this screen.

Adjust Dual Gantry Skew

This soft key moves the master rail motor (by using the directional manual jog arrow keys) to correct or realign gantry skew, after you enter the Machine Setups password.

Send Tilt Rotator Home

This soft key moves the bevel head to the predefined home position.

Move Rotate Axis

This soft key moves to a rotate axis position that you enter.

Move Tilt Axis

This soft key moves to a tilt axis position that you enter.

Space Torches

This soft key executes the torch spacing routine. When you choose this soft key, Phoenix generates a .txt file and moves the torches to evenly spaced positions along the transverse axis.

Note: This feature requires special program codes. See the EDGE Connect Programmer Reference Manual (809550) for part program codes.
