Pause

To pause cutting, do one of the following actions:

  • Press the Stop button on the operator console.
  • Press F10 on the keyboard.
  • Press Stop on the Soft Op Con.

 

    

The Cut Loss Recovery features of the CNC show on the Pause screen. The Pause screen displays when you press Stop or if the Cut Sense input turns off. Choose the Cancel soft key on the Pause screen to cancel the part program.

Soft keys

Return to Start

This soft key returns to the start point of the part program. If you use the Return to Start function after a cut loss, the current position of the cutting device on the path is lost.

Backup and Forward on Path

Use these two soft keys to move backward and forward along the cut path. Press the Start key to resume the cut at the program cut speed. The Backup and Forward on Path soft keys also allow full movement through all segments of a Shape Repeat part.

Backup and Forward on Path use the Move Speed displayed on the Pause screen. Choose the Change Move Speed to speed up or slow down the Move Speed.

Move to Pierce

Choose the Move to Pierce soft key to position the tool at a pierce point.

Change Cut Mode

This soft key alternates the restart mode between Cut and Trial. This allows the operator to move through the part partially as a trial cut and partially as an actual cut.

Change Move Speed Cycles through the four move speeds that are available: Maximum Machine Speed, High Jog Speed, Medium Jog Speed and Low Jog Speed from the Speeds setups.

On-Path Restart

To restart the cut at the pierce point that you selected using On-Path Backup, press the Start key. The cut speed and the cut mode are the same as they were before the motion was paused unless the values have been edited in the Watch Window.

While the Pause window displays, the manual arrow keys are fully functional so you can move the cutting device. This allows you to move the machine in any direction (not necessarily along the path) to inspect the partially cut piece. Once the cutting device is moved off the cut path, the Off-Path Pause window displays.

Return to Path

Press this soft key in the Off-Path Pause window to return the cutting device to the point on the cut path from which it was jogged away. This feature is useful for inspecting or replacing components after cut loss and then returning to the cut loss point. When the cutting device is back on the cut path, the On-Path Pause window is restored and cutting can resume.

Move Part moves the entire part on the plate. The point along the cut path to which the cutting device moves becomes the current position of the cutting device. The On-Path Pause window displays again because the cutting device is on path.

Off-Path Restart

Press the Start key from the Off-Path Pause window, to construct a lead-in from the off-path point back to the original part.

In a cut loss situation, the operator can use the Backup on Path key in the On-Path Pause window menu to position the cutting device on the cut path where the cut was lost. The operator can then use the manual arrow keys to jog the cutting device off path to a suitable pierce point.

Press Start at this point to cut a new lead-in from the off-path pierce point to the point along the path from which the cutting device was jogged away. When the cutting device is back on path, it continues along the path to cut the remainder of the part.

Rush Job Interrupt

This soft key allows you to pause the current part program and retain the part and current position information. On the Pause screen, press the Cancel key. A prompt displays on the screen to allow you to save the part information.

If you select Yes, the Resume Last Part button displays on the Files screen. You can load and execute another part program and then return to the original part using the Resume Last Part soft key. The part program and position resumes.
