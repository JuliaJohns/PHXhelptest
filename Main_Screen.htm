Main

The Main screen is the first screen you see when you start the CNC.

The 8 keys along the bottom of the screen are called soft keys. The soft keys correspond to the function keys on a PC keyboard. The OK and Cancel soft keys let you save or cancel changes that you make in a screen.

The features shown on each screen depend on the user level (Beginner, Intermediate, or Advanced) and the features enabled on the Special Setups, Station Configuration, Watch Window screens. This and other Phoenix Help topics assume the CNC is in Advanced mode and shows all features.

The Preview window shows the current part program and its dimensions. Phoenix shows the name of the part program under this window, as well as status messages (in blue), such as “with True Hole Technology” if the part program uses this feature. Watch for the blue status messages that convey information about the part you are cutting.

Soft Keys

Help

This soft key opens the Phoenix Help window and displays information about the current screen.

Cutting Tips

This soft key displays cutting tips for the selected process.

Technical Documentation

This soft key opens the PDF Viewer and shows the EDGE Connect Manual. You can open additional manuals by selecting the soft keys at the bottom of the PDF Viewer.

CutPro® Wizard

This soft key automates the sequence of choices and selections required for cutting parts. If you have parts, nests, and cutting processes stored on your system, you can use the CutPro Wizard to simplify cutting operations. The CutPro Wizard keeps track of and displays your progress at the bottom of the wizard window.

Remote Help

This soft key opens Remote Help. Remote Help is an Internet-based diagnostic tool that allows a remote technician and a CNC operator to work together to troubleshoot issues with the CNC, plasma system, or cutting table. Using Remote Help, diagnosis and repair can often be accomplished without an on-site visit from a technician. This means that cutting machines can be returned to operation quickly and without travel and downtime.

Remote Help uses a screen-sharing tool to allow a technician to securely view a CNC over the Internet and assist the CNC operator in solving problems. Hypertherm Technical Service uses the Bomgar™ screen sharing tool, but you can choose your own screen sharing tool. Other screen sharing tools that are compatible with Hypertherm CNCs are TeamViewer® and ScreenConnect®. Skype® for Business is compatible with Hypertherm CNCs that have Windows® 10 or higher operating system. You must purchase the screen sharing tool to use with Remote Help.

Manual

This soft key opens the Manual screen for rip cutting, homing axes, or accessing manual options such as laser pointer offset or torch spacing.

Shape Manager

This soft key opens the Shape Manager screen where you can load a simple shape, use the text editor or Shape WizardTM to edit a part, or open ProNest® CNC.

Files

This soft key opens the Files screen where you can load, save, download or upload part files. This soft key's default action is to show the "Load from Disk” actions.

Current Part Options

This soft key lets you scale, rotate, mirror, and repeat the part.

Setups

This soft key opens the Cutting screen. The Cutting screen gives you access to the Process, Watch Window setup, Diagnostics, and the password-protected setup screens.

View Part/View Sheet

This soft key toggles the display of the part in the Preview window.

  • View Part zooms in to show only the part.
  • View Sheet zooms out to show the part in relationship to the entire workpiece. Phoenix shows the sheet dimensions that you entered in the Cutting screen.

Zoom +/-

This soft key appears in the Watch Window when you select the Manual Jog option. This key magnifies the part. After you zoom out, press the + key to zoom in again. Horizontal and vertical scroll bars are shown. Press the - key to zoom out.

Manual

This soft key lets you do rip cuts, home the machine axes, and other manual operations.

Scroll Bars

These soft keys shift the view of the plate horizontally and vertically. The scroll bars only appear if you are in View Sheet mode and have zoomed in on the sheet (such as with the + key on the keyboard). When not cutting, press and move a scroll bar or hold down the Shift key and press the arrow keys on the keyboard.

While the CNC is cutting, the view automatically shifts as the cut path reaches one of the edges of the view.

Change Cut Mode

This soft key selects trial, Oxyfuel, plasma, waterjet, and laser cut modes, depending on the tooling selected in the Special Setups screen.

Change Consumable

This soft key opens the Change Consumable screen.

Zero Positions

This soft key sets the current positions to 0 (zero) on the Transverse, Rail, and Dual Gantry axes.
