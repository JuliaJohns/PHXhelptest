Files or Load from Disk

By default, the Files soft key shows the “Load from Disk” actions. You can load part programs from the CNC hard drive, a USB memory stick, or external mapped drives (network option) into working memory on the CNC. Permission to add or remove files and folders from the hard drive is assigned in the Status/Feature list on the Special Setups screen.

Phoenix filters the file types that you see in this screen by the file extension that you enter in the Special Setups screen. For example, if you want to open CNC files, then you must enter CNC as a file extension on the Special Setups screen.

Note: Phoenix uses ProNest CNC to open DXF and DWG files. However, if these extensions have been entered in the Special Setups screen, you will still see these types of files. To open DXF and DWG files, choose the ProNest CNC soft key on the Shape Library screen.

Field Definitions

Load from

Choose a source that contains the file to load from the list.

Files

This field shows a list of files from the selected source.

Preview

Check this box to preview the selected file.

File Name

This field displays the name of the file selected in the Files list.

Soft Keys

Load from Disk

This soft key opens a screen where you can load part programs from a USB memory stick or the CNC hard drive.

Save to Disk

This soft key opens a screen where you can save a part program to a USB memory stick or the CNC hard drive.

Resume Last Part

This soft key loads and executes another part program and then returns to the original part program. The part program and position resumes.

Show Certain Files

This soft key lets you use wildcard searches, both asterisk (*) and question mark (?), to search the selected folder for specific part files.

Show All Files

This soft key lets you view all files with the file extensions selected in the Special Setups screen, rather than view only the selected files extensions.
