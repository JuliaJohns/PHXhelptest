Manual

Use the Manual screen for rip cutting, homing axes, or accessing manual options such as laser pointer offset or torch spacing. To open the Manual screen, choose any of the following:

  • Manual soft key on the Main screen
  • Manual soft key on the Soft Op Con
  • F11 on a keyboard

 

Notes:

  • If Phoenix shows a field bus error, press the Manual soft key to restart the EtherCAT network.
  • To activate Latch Manual Keys (enabled on the Special Setups screen), press the Manual key twice.

Soft Keys

Return to Start

When you open the Manual screen, Phoenix saves the Transverse and Rail positions. After rip cutting or other manual operations, use Return to Start to return to this position.

Move/Mark/Cut Distance

The name of this soft key changes depending on the manual mode you select with the Change Manual Mode soft key.

Use Move Distance to move to a specific X/Y location by entering a Transverse and Rail distance.

Use Cut Distance to create a rip cut from the current torch location. Rip Cut mode makes a cut along a specified linear path. Motion stops and cutting action ceases when the new position is reached or when the STOP key is pressed.

Manual Options

This soft key opens the Manual Options screen so that you can set up a laser pointer or use torch spacing.

Home Axes

This soft key opens the Home Axes screen where you can set up to 12 home positions and home all of the axes.

View Part/View Sheet

This soft key toggles the display of the part in the Preview window.

  • View Part zooms in to show only the part.
  • View Sheet zooms out to show the part in relationship to the entire workpiece. Phoenix shows the sheet dimensions that you entered in the Cutting screen.

Change Manual Mode

This soft key cycles through the Move Only, Rip Cut, and Rip Mark modes.

Change Move Speed

This soft key cycles through the following four Move Speeds from the Speeds screen:

  • maximum machine speed

  • high jog speed

  • medium jog speed

  • low jog speed

Zero Positions

This soft key returns all axes positions to 0 (zero).
