Speed 0 to

This setting sets speed ranges (or breaks) and lets you set the acceleration rate for each speed break. When you enter a speed break that is lower than the maximum machine speed, Phoenix creates a new speed break at the maximum machine speed. You can set up to 5 speed ranges.

Acceleration rate

This setting sets both the acceleration and deceleration rate for all motion.

The higher the acceleration rate, the more quickly the cutting machine will reach program speed. The lower the acceleration rate, the more smoothly the machine will position the cutting device.

Enter a value in milliGs for the desired acceleration rate. One milliG is approximately 0.386 inches per second2. A value of 10 milliGs represents a starting point from where you can fine tune the acceleration rate for your cutting machine.

Peak S curve acceleration

This field shows the maximum acceleration used during the beginning of a cut and represents a rate of acceleration that is 50% greater than the average rate of acceleration. This parameter appears when S Curve Acceleration (at the lower left in the Speeds screen) is set to Yes.

Fast deceleration rate

This parameter is activated by the Fast Stop or Torch Collision inputs and specifies in milliGs how quickly the system stops when the Fast Stop input activates. Generally, this value is much higher than the acceleration rate.

Maximum machine speed

Enter the maximum value at which the system is capable of contouring. This parameter limits the range of all subsequent speed entries. In addition, it scales the speedometer in the Watch Window during runtime operation.

The maximum speed at which the CNC can contour can be calculated with the following equation.

Maximum contouring rate (in/min or mm/min) = 60 seconds x 1 inch x 100,000 edges

High, medium, and low jog speeds

Enter preset jog speeds for use in manual motion for rip cutting or manual moves. Phoenix uses these preset speeds on the Manual screen when you choose the Move Speed soft key.

Fast and Slow home speeds

Phoenix uses the Fast home speed when it moves the cutting device to the home or limit switch until it contacts the switch.

Phoenix uses Slow home speed when it moves the cutting device off of the home or limit switch and proceeds to the marker pulse. When Phoenix detects the marker pulse, it decelerates to a stop and performs a small move in the opposite direction to the marker pulse.

Minimum corner speed

This value is the minimum speed to use when moving into and out of corners. In some situations, a minimum corner speed prevents over-burning at corners. For most cutting jobs, set this parameter to 0 (zero).

Limited machine speed

This value sets the machine speed when the Limited Speed input is active for increased safety during machine testing or maintenance.

Creep speed

This value sets the percentage of the maximum machine speed (in general, 25%). Phoenix uses Creep after piercing, typically in a lead-in, before the machine accelerates to the cut speed.

Torch height disable speed

This value specifies the percentage of the program cut speed at which the CNC disables the height control.

For example, when the cutting machine slows to cut a part feature such as a corner, the arc voltage increases. Increased arc voltage causes the torch to move toward the workpiece. When the cutting machine slows down to this percentage of the cut speed, the CNC disables THC voltage tracking.

See Torch height disable for more information.

Distance before THD Speed

This value specifies the distance before turning a corner at which the CNCs disables the torch height control. This distance is active whenever the program cut speed drops below the Torch Height Disable Speed, or when segments intersect at an angle greater than the tangent angle.

Distance after THD speed

This value specifies the distance after turning a corner to enable voltage tracking for torch height control. This distance is active whenever the speed drops below the Torch Height Disable Speed, or when segments intersect at an angle greater than the tangent angle.

Limited speed if not homed

This value sets the machine speed when the Limited Speed input is active and the cutting machine has not been homed.

THC acceleration rate

This value specifies the acceleration rate for smooth and stable movement of the Sensor THC. The higher the acceleration rate, the quicker the THC reaches cutting speed. The lower the acceleration rate, the smoother the machine will position the cutting device. Values are in milliGs.

Maximum THC speed

This value specifies the maximum speed for the Sensor THC in mm/min or in/min.

THC jog speed

This value specifies speed when moving the lifter up or down using the jog keys on the CNC or on the screen.

THC home/Fast IHS speed

This value specifies the speed that is used during the following actions:

  • Home the torch at the top of the slide.

  • Move the torch from the selected IHS height until it senses the workpiece.

THC slow IHS speed

This value specifies the speed used to retract the torch after it senses the workpiece. The torch retracts at this speed until contact sense is lost. This determines the exact distance to the workpiece regardless of any flexing of the workpiece. A very low speed is recommended for optimal accuracy.

Rotate acceleration rate

This value specifies the acceleration rate, in revolutions per second2, for smooth bevel head motion. The higher the acceleration rate, the quicker the Rotate axis reaches cutting speed. The lower the acceleration rate, the smoother the machine will position the cutting device.

Maximum rotate speed

This value specifies the fastest speed for the rotate axis rotation in revolutions/minute.

Rotate fast jog/home speed

This value specifies the fastest speed for the rotate axis when it jogs.

Rotate slow jog/home speed

This value specifies the slowest speed for the rotate axis when it jogs.

Tilt acceleration rate

This value specifies the acceleration rate, in revolutions/second2, for smooth and stable movement of the bevel head. The higher the acceleration rate, the quicker the tilt axis reaches cutting speed. The lower the acceleration rate, the smoother the machine positions the cutting device.

Maximum tilt speed

This value specifies the fastest speed for the tilt axis, in revolutions/minute.

Tilt fast jog/home speed

This value specifies the fastest speed for tilt axis when it jogs.

Tilt slow jog/home speed

This value specifies the slowest speed for tilt axis when it jogs.

S-curve acceleration

Select Yes to enable S-curve acceleration. Select No to use trapezoidal acceleration. Depending on the application and table mechanics, S-curve acceleration can reduce or eliminate “jerk” that results from trapezoidal motion. S-curve motion is available in the Advanced Plasma/Oxyfuel and Advanced Waterjet options to the EDGE Connect.

See S-curve acceleration for more information.

Caution! To prevent machine damage

If the existing trapezoidal acceleration parameter value is set to the maximum acceleration capability of the cutting table, reduce this acceleration parameter by 50% before enabling the S-curve feature.
