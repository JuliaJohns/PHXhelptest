Shape Wizard

Shape WizardTM is a part editor that provides a user-friendly, graphical interface for editing part programs. You can view the segment type and the motion commands associated with the segment.

Navigation: Main screen > Shape Manager > Shape Wizard

With a part program displayed on the Main screen:

  • The part shows in the Preview window and the corresponding code shows in the EIA Text window.
  • As you edit lines of code, the changes show in the Preview window.
  • If you don’t know EIA RS-274D codes, you can edit or create segments by making entries and selections in the Segment Data fields below the EIA Text window.
  • Zoom keys decrease or increase the size of the part in the Preview window.

Soft Keys

Shape Options

This soft key opens the Current Part Options screen where you can apply kerf, set up part repeats, and change the scale, change the rotate angle, or mirror the part.

Replace Segment

This soft key replaces the segment highlighted in gray in the Text Editor window with the segment selected from the Segment Type window.

Insert Before Segment

This soft key inserts the segment selected from the Segment Type window to be inserted before the segment highlighted in gray in the Text Editor window.

Insert After Segment

This soft key inserts the segment selected from the Segment Type window after the segment highlighted in gray in the Text Editor window.

Remove Segment

This soft key deletes the segment that is highlighted in gray or blue in the EIA Text window from the part program.

View Part/View Sheet

This soft key toggles the display of the part in the Preview window.

  • View Part zooms in to show only the part.
  • View Sheet zooms out to show the part in relationship to the entire workpiece. Phoenix shows the sheet dimensions that you entered in the Cutting screen.
