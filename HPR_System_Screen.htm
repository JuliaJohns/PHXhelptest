Field Definitions

Line Voltage

This field is the level of incoming power to the power supply.

Current Setpoint

This field is the amperage set for the power supply.

Coolant Flow

This field is the rate at which coolant is flowing.

State

This field is the active state of the power supply.

Error

This field is the code and description identifying the system error that has occurred, if any. For a more detailed description of each error and for possible corrective actions you can take, see the power supply instruction manual.

Arc On Time

This field is the time that the power supply has been producing an arc.

System On Time

This field is the time the power supply has been ON.

Total Starts

This field is the number of arc transfers.

Total Start Errors

This field is the number of times the arc failed to transfer.

Total Ramp Errors

This field is the number of times the power supply failed to ramp-down current.

Power Supply

This field is the plasma power supply software revision number.

Gas Console

This field is the gas console software revision number.

Chopper 1

This field is the temperature reading for the chopper, coolant, transformer, and inductors. If this temperature exceeds a maximum threshold, it will display in red. When this occurs, the power supply cannot be operated until the condition is resolved.

Coolant

This field is the temperature reading for the coolant. If this temperature exceeds a maximum threshold, it will display in red. When this occurs, the power supply cannot be operated until the condition is resolved.

Transformer

This field is the temperature reading for the transformer. If this temperature exceeds a maximum threshold, it will display in red. When this occurs, the power supply cannot be operated until the condition is resolved.

Plasma (Type)

This field is the plasma gas type.

Shield (Type)

This field is the shield gas type.

Plasma (Pressure)

This field is the plasma gas pressure. The set value shows the target gas pressure set in the power supply. The measured value displays as zero by default, but you can watch the values in this field to monitor the plasma gas pressure when you activate the diagnostic modes on the screen.

Shield (Pressure)

This field is the shield gas pressure. The set value shows the target gas pressure set in the power supply. The measured value displays as zero by default, but you can watch the values in this field to monitor the shield gas pressure when you activate the diagnostic modes on the screen.

Soft Keys

Test Preflow and Test Cutflow

Use these tests to bring up the inlet gas pressures to the recommended level. These tests can also help determine if the gas system can sustain the recommended pressure levels.

Test Gas Console

This soft key performs automated tests for the autogas console. Contact an authorized service agent for use of these tests.

Coolant Override

This soft key overrides a coolant error and tests the coolant pump. This is useful for bypassing the error and purging air bubbles from the coolant line when power is turned ON.

Inputs

This soft key opens the Inputs screen for the plasma power supply or gas console.

Outputs

This soft key is password protected and opens the Outputs screen for the plasma power supply or gas console. Operating these outputs can initiate the real-time signal that could, for example, fire the torch.

HPR Information

This soft key opens the HPR Information screen.
