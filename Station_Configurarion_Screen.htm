Station Configuration

The Station Configuration screen lets you assign the process to each station in the cutting system. In this screen, you select the type of torch height control and the model of the tool (plasma or marker) for the station. When you select the torch height and tool, Phoenix customizes the Process screen to your selections and enables the cut charts for the tool. Phoenix also displays the instruction manual for a Hypertherm plasma system when you choose the Technical Documentation soft key on the Main screen.

To open the Station Configuration screen, choose Setups > Password > Station Configuration.

Guidelines for using the Station Configuration screen

Follow these general guidelines to determine whether you need to make selections in the Station Configuration screen:

  • When using the Sensor THC for a lifter, assign the Sensor THC to an axis on the Machine Setups screen first, and then select it as a lifter in this screen. Torch height control parameters specific to the Sensor THC lifter become available in the Process screen so that you can optimize cutting performance.
  • Select the process used by the tool on the station. For example, when you select HPR for Plasma 1, the HPR cut charts become available for the process.
  • When you select an HPR or XPR, Phoenix automatically assigns fixed function I/O for the power supply. This I/O is assigned above number 512. It can be viewed only in the Watch Window.
    Note: Phoenix currently does not support an HPRXD plasma power supply and an XPR plasma power supply on the same cutting system.
  • After setting up the stations and exiting the Station Configuration screen, the stations appear in the Soft Op Con and Phoenix automatically assigns the Station I/O to I/O points 129 and above.
  • Oxyfuel systems DO NOT require a selection in the Station Configuration screen. Oxyfuel cut charts are available after you select Oxyfuel as a Tool Installed in the Special Setups screen (Setups > Password > Special Setups).
  • Use the soft keys to open the Station Configuration screen for Stations 1 – 2, 3 – 4, 5 – 6, 7 – 8.
  • Use the Reset All soft key to return all settings for all stations to None.
