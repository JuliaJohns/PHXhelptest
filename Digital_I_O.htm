Digital I/O

Use the Digital I/O screen to assign digital inputs and outputs. Phoenix supports up to 512 digital inputs and 512 digital outputs.

General purpose I/O

Use general purpose I/O (sometimes called generic I/O) for any cutting process to control both of the following:

  • Cutting machine hardware (such as overtravel switches)
  • A single cutting station

Select the I/O points from the Inputs or Outputs list and assign the function to the I/O point. On the EtherCAT network, the first I/O point is found on the first drive on the network. If Inputs 1 – 7 are on drive 1, then 8 – 14 are on drive 2, and so on. An I/O module follows in physical order after the drives (but before the plasma power supply). If there are 4 drives in the network, then the I/O module digital inputs would start with 29.

In Phoenix, general purpose I/O is 1 – 512. Station I/O, which is automatically assigned when you configure a station in the Station Configuration screen, starts at I/O 129 and above.

Numbered I/O

Use numbered I/O to control multiple cutting stations. For example, assign the outputs Station Enable 1 and Cut Control 1 to Station 1 and assign the outputs Station Enable 2 and Cut Control 2 to Station 2, and so on.

Fixed Function I/O

Phoenix assigns fixed function I/O automatically when you select an HPR in the Station Configuration screen. Fixed function I/O for the HPR is not shown as assigned to any I/O points in the Inputs or Outputs lists on the I/O screen and does not appear in the I/O Diagnostics screen. You can view it in the Watch Window or Oscilloscope.

Phoenix also assigns fixed function I/O to the factory-installed operator console on the EDGE Connect TC.

Fixed function I/O for HPRXD plasma supplies

Phoenix automatically assigns the I/O for HPRXD power supplies. The I/O appear in the Watch Window after input or output 512.

PLC I/O

You can assign an I/O point to the PLC Connect or PLC Connect LT by selecting the PLC checkbox. For more information, see the PLC Connect Application Note (809570).

Note: The Soft Op Con does not accurately reflect the state of I/O controlled with PLC, unless you check the Op Con Read-Only checkbox in addition to the PLC checkbox.

Op Con Read-Only

The Op Con Read-Only checkbox is useful for inputs and outputs that are controlled by external hardware switches or PLC.

If you select this box:

    •  The Soft Op Con cannot control the selected I/O.

    •  The Soft Op Con updates to reflect the state of the I/O selected with an external hardware switch or PLC.

Example: If the checkbox is selected, the Soft Op Con indicates that Manual Select 9 is turned OFF after the CNC operator successfully turns OFF Manual Select 9 with an external hardware switch or PLC.

If you do not select this box:

    •  The Soft Op Con can control the selected I/O.

    •  If an external hardware switch or PLC changes the state of the selected I/O, the state of the I/O shown on the Soft Op Con does not update.

 Example: If the checkbox is not selected, the Soft Op Con continues to indicate that Auto Select 9 is ON after the CNC operator successfully turns OFF Auto Select 9 with an external hardware switch or PLC.

Logic

The value in this field represents I/O points in the range selected (1 – 32, 33 – 64, and so on) that are set to normally closed. To see a bit-wise view of all of the inputs or outputs 1 – 32 that have been set to normally closed, you can enter the decimal number into binary or hexadecimal with a calculator. These normally-closed inputs or outputs display as a ’1’ instead of a ‘0’. The least significant bit represents Input 1 (or the lowest input or output selected) and the most significant bit represents Input 32 (or the highest input or output selected). Every time you select a different group of 32 inputs or outputs, the number updates to reflect the currently selected group of 32 inputs or outputs.

Drive Enables

This option determines how the CNC responds to drive faults. After Phoenix disables the drives, it turns off the Drive Enable output.

Series

Phoenix combines the individual fault ramp times (to a maximum of 2 seconds) before it disables the drives.

Independent

Phoenix disables each drive individually when its axis fault ramp time elapses.

Initial Feedback Delay

This option allows the initial feedback from the drive system to be delayed after initial power up. This allows the drive system to become fully enabled before it enables the position loop.

Joystick Installed

This setting is automatically enabled for EDGE Connect TC. If you are adding your own operator console, you must enable the joystick on the Digital I/O screen.
